# Zeitung-Reservierungsliste

So sehen Sie alle Reservierungen in einer Liste.

**Video:** [Zeitung Reservierungsliste](https://youtu.be/zhQWNf9jfNw)

## Schritte

1. Gehen Sie zu **Auftr. Kunden** → **Auftrag Kunden**.
2. Klicken Sie in der Werkzeugleiste des Formulars auf das Symbol für die **Selektion**.
3. Klicken Sie auf das Symbol für die **Indexliste** oder drücken Sie die Taste **F2**.

Es werden Ihnen alle Reservierungen angezeigt.
