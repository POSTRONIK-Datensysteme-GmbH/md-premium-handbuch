# Vorremission mit Barcode

So fügen Sie Objekte per Barcode/EAN einem Remissionsaufruf hinzu.

## Voraussetzung

Es muss ein **offener Remissionsaufruf** für den entsprechenden Lieferanten bestehen.

## Schritte

1. Klicken Sie auf **Zeitungen** → **Vorremission mit Barcode**.
2. Geben Sie unter **[1] RefNr** den EAN-Code ein oder scannen Sie mit dem Barcodescanner. Bestätigen Sie mit **[2] Anzeigen**.
3. Um das Objekt an einen Remissionsaufruf anzufügen:
   - **Rechtsklick** auf die Zeile des Objekts → **An Remissionsauftrag anfügen…**, **oder**
   - **Doppelklick** auf den Zeilenkopf des Objekts.

Die eingegebene Menge wird an den **jüngsten** vorhandenen Remissionsauftrag des Lieferanten am Ende angefügt.

**Hinweise:** Beim Einfügen wird das „Kommissionsdatum“ auf das aktuelle Tagesdatum gesetzt. Ein Objekt kann nicht angefügt werden, wenn es mit derselben Ausgabenummer bereits im Remissionsaufruf ist.
