# Retourenaufruf an MDE-Gerät senden

So senden Sie einen Retourenaufruf an das MDE-Gerät.

## Schritte

1. Wählen Sie den Menüpunkt **Zeitungen** → **Neue Retourenaufrufe**.
2. Machen Sie einen **Rechtsklick** auf die Zeile der Retoure und wählen Sie im Kontextmenü **AnMDE Kennzeichen setzen**.
3. Bei erfolgreicher Übertragung an das MDE-Gerät wird ein **Zeitstempel** im Feld **Datum** eingetragen.
