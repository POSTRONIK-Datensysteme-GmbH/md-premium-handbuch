# Vorremission aus Lieferscheinen

So führen Sie Vorremissionen direkt aus gelieferten Zeitungsobjekten aus.

## Schritte

1. Wählen Sie aus der Schnellstartleiste **Zeitungen** → **Vorremission aus Lieferscheinen**.
2. Wählen Sie im Bereich **Lieferdatum** einen Datumsbereich und klicken Sie auf **Anzeigen**. Es werden nur Lieferungen aus diesem Zeitraum angezeigt.
3. Im Fenster werden alle Lieferungen von Zeitungsremissionen im gewählten Zeitraum angezeigt, inkl. Liefer-, Verkaufs- und Remissionsinformationen zu den letzten 10 Ausgaben pro Objekt.
4. Markieren Sie im oberen Anzeigebereich ein **Lieferobjekt**. Im unteren Bereich wird die zugehörige Ausgaben-Mengenübersicht angezeigt.

**Objekt an Remissionsaufruf anfügen** (es muss ein offener Remissionsaufruf für den Lieferanten bestehen):

- **Rechtsklick** auf die Zeile des Objekts → **An Remissionsauftrag anfügen…**, **oder**
- **Doppelklick** auf den Zeilenkopf des Objekts.
5. Im Dialog die gewünschte **Vorremissionsmenge** eingeben (Vorschlag: Liefermenge) und mit **OK** bestätigen.

**Hinweise:** Beim Einfügen wird das „Kommissionsdatum“ auf das aktuelle Tagesdatum gesetzt (verhindert Änderung bei „Mengen auf Lagerstand setzen“). Ein Objekt kann nicht angefügt werden, wenn es mit derselben Ausgabenummer bereits im Remissionsaufruf ist.
