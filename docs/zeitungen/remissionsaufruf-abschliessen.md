# Remissionsaufruf abschließen

Wenn zu einem Remissionsaufruf alle Mengen korrekt erfasst wurden, muss der Remissionsauftrag in eine **Remission** umgewandelt werden. Die Remission wird dann automatisch an den Grossisten übermittelt.

## Schritte

1. Starten Sie den Menüpunkt **Zeitungen** → **Neue Retourenaufrufe** und öffnen Sie einen Remissionsauftrag durch **Doppelklick** auf den Zeilenkopf.
2. Kontrollieren Sie die Auftragspositionen des Remissionsauftrags auf Richtigkeit und Vollständigkeit.
3. Wählen Sie im Fenstermenü **Belegdruck** den Menüpunkt **Remission**.
4. Im Dialog wählen Sie **Bildschirm**-Druck und warten Sie auf die Ausgabe.
5. Die Remission ist nun abgeschlossen.
