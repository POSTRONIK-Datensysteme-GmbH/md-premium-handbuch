# Remissionen

Nach dem Import von Remissionsaufrufen aus den EH-2000-Daten werden Remissionsaufrufe als Belegart **AM** (Aufruf zur Remission) angelegt. Remissionsaufrufe können beliebig oft berechnet (Mengen auf Lagerstand setzen) und ausgedruckt werden. An vorhandene Remissionsaufrufe können Zeitungsartikel zur Vorremission angefügt werden – manuell oder dialoggeführt aus gelieferten Objekten (siehe [Vorremission aus Lieferscheinen](/docs/zeitungen/vorremission-aus-lieferscheinen), [Vorremission mit Barcode](/docs/zeitungen/vorremission-mit-barcode)). Nach dem Erfassen aller Retour-Mengen werden Remissionsaufrufe per Belegdruck in eine **Remission** (Auftragscode **ZM**) umgewandelt. Remissions-Belege werden automatisch an den Grossisten übermittelt.

## Remissionsaufruf öffnen und drucken

1. Wählen Sie den Menüpunkt **Zeitungen** → **Neue Retourenaufrufe**. Im Fenster werden alle Zeitungs-Remissionsaufrufe mit den Netto-Summen angezeigt.
2. Öffnen Sie einen Remissionsaufruf durch **Doppelklick** auf den Zeilenkopf. Achten Sie darauf, dass im Beleg das Register **Artikel** ausgewählt ist.
3. Drucken Sie den Beleg über das Fenstermenü **Belegdruck** → **Remissionsauftrag (AM)** aus.
4. Mit dem ausgedruckten Beleg können die Zeitungsobjekte im Verkaufs- und Lagerraum gesucht werden. Tragen Sie am Beleg die **tatsächlich gefundene Anzahl** ein. Der Beleg kann geschlossen und jederzeit wieder geöffnet werden.
