# Zeitungs-Nachbestellungen

Zeitungs-Nachbestellungen werden zuerst als „Neutraler“ Lieferantenbeleg (**AN** = Anfrage) erfasst und nach dem Erfassen aller Bestellpositionen in eine Zeitungsbestellung (**BS** = Bestellung Zeitung) umgewandelt und an den Grossisten geschickt (z. B. per E-Mail).

## Variante 1: Nachbestellung aus Lieferscheinen

1. Gehen Sie vor wie unter [Vorremission aus Lieferscheinen](/docs/zeitungen/vorremission-aus-lieferscheinen) beschrieben.
2. Im **Kontextmenü** (Rechtsklick) auf eine Lieferposition wählen Sie **An Anfrage anfügen (Zeitungs-Bestellung)**.
3. Im Mengen-Dialog die **Nachbestellmenge** eingeben.
4. Das Zeitungsobjekt wird an eine offene **Anfrage** des Lieferanten angefügt. Gibt es keine offene Anfrage, wird eine neue erstellt.

## Variante 2: Nachbestellung aus dem Artikelstamm

1. **Stammdaten** → **Artikel**. Suchen Sie die gewünschte Zeitung wie unter [Bestehende Artikel suchen nach Bezeichnung](/docs/stammdaten/bestehende-artikel-suchen-bezeichnung).
2. Wählen Sie das Register **[1] Ausgaben Info** und suchen Sie die entsprechende Ausgabe.
3. **[2]** Rechtsklick auf die Ausgabe → **[3] An Anfrage anfügen (Zeitungs-Bestellung)**. Gewünschte Menge eingeben und mit **OK** bestätigen.

## Bestellung abschließen und versenden

4. **Zeitungen** → **Neue Anfragen**. Alle offenen Anfragen der Lieferanten werden angezeigt.
5. **Doppelklick** auf den Zeilenkopf der gewünschten Anfrage – die gesamte Bestellung wird angezeigt.
6. Im Fenstermenü **Belegdruck** → **Bestellung**. Im Dialog das Bestelldatum bestätigen (z. B. aktuelles Tagesdatum).
7. Im Druckdialog: **Ausdruck** auf Papier oder **Export** (E-Mail). Für E-Mail **Export** wählen.
8. Im folgenden Dialog die E-Mail-Nachricht prüfen und mit **Senden** versenden. Mit **OK** bestätigen (die vorgeschlagene E-Mail-Adresse sollte stimmen).
