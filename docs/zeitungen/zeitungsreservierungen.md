# Zeitungsreservierungen

So erfassen Sie Zeitungsreservierungen für Kunden.

**Video:** [Zeitungsreservierungen](https://youtu.be/tbyTQFWc4b4)

## Schritte

1. Gehen Sie zu **Auftr. Kunden** → **Auftrag Kunden**.
2. Klicken Sie in der Werkzeugleiste des Formulars auf das Symbol **Neu**. Ein neuer Kundenbeleg wird erzeugt, der Cursor springt ins Feld **Matchcode**.
3. Geben Sie die ersten Buchstaben des Kunden ein (z. B. „MUSTERMANN“) und drücken Sie **ENTER** oder klicken Sie auf **…** neben dem Feld Matchcode. Es wird eine Liste mit passenden Kunden angezeigt. (Die Liste erscheint nur bei mehreren Treffern.)
4. Wählen Sie den Kunden mit **ENTER** oder Doppelklick. Speichern Sie den Kundenbeleg mit dem **Diskettensymbol** oder **F10**.

### Artikelpositionen hinzufügen

5. Klicken Sie in der Werkzeugleiste der Positionsliste auf das Symbol **Neuer Datensatz**. Das Feld **RefNr** wird angesprungen.
6. Im Feld **RefNr** geben Sie eine Zeitungsobjektnummer oder einen Barcode ein (oder scannen Sie). **ENTER** oder **…** – bei mehreren Treffern den richtigen Artikel wählen.
7. Erfassen Sie im Feld **Menge** die gewünschte Menge und ggf. die Gratis-Menge.
8. Speichern Sie die Position mit dem **Diskettensymbol** in der Werkzeugleiste.
9. Klicken Sie unter **Belegdruck** **[1]** auf **Zeitungs-Reservierung Liste** **[2]**.
10. Die Zeitungsreservierung kann ausgedruckt (**OK** **[1]**) oder auf dem **Bildschirm** **[2]** angezeigt werden. Anzeige mit **X** schließen.

## Wichtig

Nach der Übernahme der Zeitungslieferscheine muss der Punkt **EH2000 Zeitungsreservierungen** täglich ausgeführt werden. Finden Sie ihn nicht in der Schnellstartleiste: **Datenpflege** → **EH2000** → **EH2000 Zeitungsreservierungen**.
