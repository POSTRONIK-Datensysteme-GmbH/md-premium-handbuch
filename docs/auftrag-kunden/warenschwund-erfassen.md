# Warenschwund erfassen (Diebstahl, Schwund, Verderb)

So erfassen Sie Warenschwund in MD-Premium (Diebstahl, Schwund, Verderb) über einen Kundenbeleg mit dem internen Kunden.

## Schritte

1. Gehen Sie in den Bereich **Auftr. Kunden** und wählen Sie **Auftrag Kunden** aus.
2. Klicken Sie auf das Symbol **Neu** in der Werkzeugleiste des Formulars. Ein neuer Kundenbeleg wird erzeugt und in einem leeren Formular angezeigt.
3. Geben Sie **INTERN** im Feld **Matchcode** ein und bestätigen Sie mit **ENTER**.
4. Speichern Sie den neu erstellten Kundenbeleg mit dem **Diskettensymbol** oder mit **F10**.

### Artikelpositionen hinzufügen

5. Klicken Sie im Positionslistenbereich auf das Symbol **Neuer Datensatz**. Eine neue Artikelposition wird erzeugt, das Feld **RefNr** wird angesprungen.
6. Geben Sie die Lieferantenbestellnummer oder den Barcode im Feld **RefNr** ein (oder scannen Sie), und wählen Sie den gewünschten Artikel aus der Liste aus.
7. Erfassen Sie die gewünschte **Menge** im Feld Menge und speichern Sie die Artikelposition mit dem **Diskettensymbol**.
8. Wiederholen Sie die Schritte 5–7 für alle weiteren Positionen.
9. Klicken Sie unter **Belegdruck** auf **Abgang sonstiges**.

Unter **Alle Berichte** → **OLAP Abgang Sonstige** finden Sie eine Auswertung über die als Warenschwund erfassten Positionen.
