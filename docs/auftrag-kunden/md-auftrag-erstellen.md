# MD Auftrag erstellen

So erstellen Sie einen Kundenauftrag und fügen Artikelpositionen hinzu (z. B. für Zeitungs-Reservierung).

## Schritte

1. Wählen Sie im Menü **Auftr. Kunden** → **Auftrag Kunden**. Der zuletzt bearbeitete Kundenbeleg wird angezeigt. Klicken Sie auf das Symbol **Neuer Datensatz** in der Werkzeugleiste.
2. Klicken Sie auf den Button **…** rechts neben dem Feld **Matchcode** und wählen Sie den gewünschten Kunden aus der Liste aus.
3. Speichern Sie den Kundenbeleg mit dem **Diskettensymbol** oder **F10**.

### Artikelpositionen hinzufügen

4. Klicken Sie auf das Symbol **Neuer Datensatz** in der Werkzeugleiste der **Positionsliste**.
5. Im Feld **RefNr** geben Sie eine Lieferanten-Artikelnummer oder einen Barcode ein (oder scannen Sie). Wählen Sie den gewünschten Artikel aus der Liste aus.
6. Erfassen Sie die gewünschte **Menge** im Feld Menge.
7. Speichern Sie die Position mit dem **Diskettensymbol** oder **F10**.
8. Wiederholen Sie die Schritte 4–7 für weitere Positionen.

### Zeitungs-Reservierung

9. Klicken Sie unter **Belegdruck** auf **Zeitungs-Reservierung Auftrag** und bestätigen Sie das Fenster mit **OK**.
