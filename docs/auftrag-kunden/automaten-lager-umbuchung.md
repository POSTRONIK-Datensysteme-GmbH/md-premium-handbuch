# Automaten-Lager-Umbuchung

So buchen Sie Artikel in ein Automatenlager um.

## Schritte

1. Wählen Sie **Auftr. Kunden** → **Auftrag Kunden**. Klicken Sie in der Werkzeugleiste auf das Symbol **Neuer Datensatz**. Ein neuer Kundenbeleg wird erzeugt, der Cursor springt ins Feld **Matchcode**.
2. Klicken Sie auf **…** neben dem Feld **Matchcode**. Wählen Sie den gewünschten **Automaten** (Kundenbeleg) mit **ENTER** oder Doppelklick aus.
3. Klicken Sie auf **…** neben dem Feld **Ziellager**. Wählen Sie das gewünschte **Automatenlager** mit **ENTER** oder Doppelklick aus.
4. Speichern Sie den Kundenbeleg mit dem **Diskettensymbol** oder **F10**.

### Artikelpositionen hinzufügen

5. Klicken Sie in der Werkzeugleiste der Positionsliste auf das Symbol **Neuer Datensatz**. Das Feld **RefNr** wird angesprungen.
6. Im Feld **RefNr** geben Sie eine Lieferanten-Artikelnummer oder einen Barcode ein (oder scannen Sie). Bei mehreren Treffern den richtigen Artikel wählen.
7. Erfassen Sie die gewünschte **Menge**.
8. Speichern Sie die Position mit dem **Diskettensymbol** oder **F10**. Erfassen Sie alle weiteren Positionen auf die gleiche Weise.
9. Nach dem Erfassen aller Positionen: Unter **Zusätze** **[4]** auf **Positionen freigeben** **[5]** klicken.
10. Unter **Belegdruck** **[1]** auf **Lagerumbuchung** **[2]** klicken. Das sich öffnende Fenster mit **OK** bestätigen.
