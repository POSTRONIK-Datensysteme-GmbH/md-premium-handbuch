# Ausdrucken von Kunden-Rechnungen

So drucken Sie Kundenrechnungen (Sammelkassenbeleg) aus oder senden sie per E-Mail.

## Schritte

1. Wählen Sie im Menü **Auftr. Kunden** → **Auftrag Kunden**, um den zuletzt bearbeiteten Kundenbeleg anzuzeigen.
2. Wählen Sie **Selektion** → **KA Belege** und filtern Sie nach **Kunden-ID** und **Datum**.
3. Geben Sie die **Kunden-ID** (6 Ziffern) und den gewünschten **Datumzeitraum** ein und bestätigen Sie mit **OK**.

**Hinweis Kunden-ID:** Wird der Kunde an der Kasse mit der Nummer 21 boniert, ist die Kunden-ID **200021**; bei Kasse 101 ist die Kunden-ID **200101**.

4. Wählen Sie **Belegdruck** → **Sammelbelegdruck** → **Sammelkassenbeleg**.
5. Drucken Sie den Sammelbeleg aus oder senden Sie ihn per E-Mail (sofern eine E-Mail-Adresse hinterlegt ist) durch Klick auf **Export**. (Bei Outlook öffnet sich eine E-Mail-Nachricht.)
