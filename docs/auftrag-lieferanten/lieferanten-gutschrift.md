# Lieferanten-Gutschrift

So erstellen Sie eine Lieferanten-Gutschrift.

**Video:** [Lieferanten-Gutschrift](https://youtu.be/tJLJ4UsYTuE)

## Schritte

1. Erfassen Sie den Lieferantenbeleg wie in der Anleitung [Lieferantenbeleg erfassen](/docs/auftrag-lieferanten/lieferantenbeleg-erfassen) beschrieben.
2. Wählen Sie im Fenstermenü **Belegdruck** → **Gutschrift**.
3. Klicken Sie auf **Bildschirm**, um die Gutschrift abzuschließen.
