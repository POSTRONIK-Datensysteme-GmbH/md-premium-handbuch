# Tobaccoland – Bestellung online senden

So übermitteln Sie eine Bestellung an Tobaccoland.

## Schritte

1. Wählen Sie im Menü **Auftrag Lieferanten** eine Bestellung von Tobaccoland aus.
2. Unter **Zusätze** finden Sie den Menüpunkt **Tobaccoland Bestellung 2019**. Klicken Sie darauf – das externe Programm **PTTobOrder** für die Übermittlung an Tobaccoland wird geöffnet.
3. Klicken Sie auf **SENDEN**, um die Bestellung an Tobaccoland zu übermitteln.

Sie erhalten anschließend eine Bestätigungs-E-Mail von Tobaccoland.
