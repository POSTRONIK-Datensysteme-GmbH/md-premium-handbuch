# Lieferantenlieferschein erfassen

So erstellen Sie einen Lieferantenlieferschein aus einem Lieferantenbeleg.

**Video:** [Lieferantenlieferschein erfassen](https://youtu.be/E23-q_FlHWk)

## Schritte

1. Erfassen Sie den Lieferantenbeleg wie in der Anleitung [Lieferantenbeleg erfassen](/docs/auftrag-lieferanten/lieferantenbeleg-erfassen) beschrieben.
2. Klicken Sie mit der linken Maustaste auf das Feld **Zusätze**. Es öffnet sich ein Fenster. Wählen Sie **Position freigeben**.
3. Klicken Sie unter **[1] Belegdruck** auf **[2] Lieferantenlieferschein**. Es öffnet sich ein Fenster.
4. Tragen Sie unter **[3] Fremdbelegsnummer** die Lieferscheinnummer ein und bestätigen Sie mit **OK**. (Bei Bedarf kann das Lieferdatum noch geändert werden.)
5. Zum Ausdrucken: **[1] OK** klicken. Wenn Sie den Lieferschein nicht drucken möchten, muss er einmal auf **[2] Bildschirm** angezeigt werden (Anzeige mit **X** schließen).
