# Moosmayr – Bestellung online senden

So übermitteln Sie eine Bestellung an Moosmayr.

## Schritte

1. Wählen Sie im Menü **Auftrag Lieferanten** eine Bestellung von Moosmayr aus.
2. Unter **Zusätze** wählen Sie **Moosmayr Bestellung Online**.
3. Es erscheint ein Pop-up-Fenster mit der Bestellnummer.

**Variante 1:** Die Bestellung ist durchgegangen.

**Variante 2:** Wird in der Meldung nach „|live|“ ein **negativer Wert** mit einer Bestellnummer angezeigt, konnte diese Bestellung nicht durchgeführt werden. Die angezeigte Bestellnummer kann bei Moosmayr nicht bestellt werden – **diese Position muss aus der Bestellung gelöscht werden**, damit Sie die Bestellung verschicken können.
