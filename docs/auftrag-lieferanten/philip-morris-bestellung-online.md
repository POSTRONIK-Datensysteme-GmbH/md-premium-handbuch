# Philip Morris – Bestellung online senden

So übermitteln Sie eine Bestellung an Philip Morris.

## Schritte

1. Wählen Sie im Menü **Auftrag Lieferanten** eine Bestellung von Philip Morris aus.
2. Unter **Zusätze** finden Sie den Menüpunkt **Philip Morris Bestellung**. Klicken Sie darauf – das externe Programm **PTPmiOrder** für die Übermittlung an Philip Morris wird geöffnet.
3. Klicken Sie auf **SENDEN**, um die Bestellung an Philip Morris zu übermitteln.

Sie erhalten anschließend eine Bestätigungs-E-Mail von Philip Morris.
