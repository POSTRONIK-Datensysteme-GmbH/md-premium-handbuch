# Offene Lieferantenlieferscheine übernehmen

So buchen Sie gelieferte Ware aus offenen Lieferscheinen ins System ein.

**Video:** [Offene Lieferantenlieferscheine übernehmen](https://youtu.be/BQmeCZ1I6VA)

## Schritte

1. Klicken Sie in der Menüleiste auf **[1] Offene Lieferscheine**. Es öffnet sich ein Fenster mit allen offenen Lieferscheinen der Lieferanten.
2. Doppelklicken Sie auf den **[2] Zeilenkopf** des gewünschten Lieferscheins. Der gesamte Lieferschein wird angezeigt. Vergleichen Sie die Rechnung vom Lieferanten mit dem angezeigten Lieferschein.
3. **Hinweis:** Überprüfen Sie die Menge der Artikel **[1]** und die Netto-EKP-Summe **[2]** mit der Rechnung. Bei Abweichungen tragen Sie die tatsächlich gelieferte Menge in der Spalte **Menge** ein und speichern die Änderungen.
4. Klicken Sie auf **Belegdruck** **[1]** und wählen Sie **Rechnung** **[2]** aus. Im sich öffnenden Fenster tragen Sie die Rechnungsnummer unter **Fremdbelegsnummer** **[3]** ein und bestätigen mit **OK**. (Das Rechnungsdatum kann ggf. noch geändert werden.)

Die Ware ist nun gebucht. Zur Kontrolle: Im Feld **[1]** sollte das Kürzel **ZR** (Rechnung) stehen.
