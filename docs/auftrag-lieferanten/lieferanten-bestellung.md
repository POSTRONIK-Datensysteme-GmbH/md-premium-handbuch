# Lieferanten-Bestellung

So erstellen Sie eine Lieferantenbestellung und drucken sie aus.

**Video:** [Lieferanten-Bestellung](https://youtu.be/T7lyD6STjRw)

## Schritte

1. Erfassen Sie den Lieferantenbeleg wie in der Anleitung [Lieferantenbeleg erfassen](/docs/auftrag-lieferanten/lieferantenbeleg-erfassen) beschrieben.
2. Klicken Sie auf **[1] Belegdruck** und wählen Sie **[2] Bestellung** aus.
3. Im sich öffnenden Fenster können Sie das Bestelldatum eingeben.
4. Bestätigen Sie mit **Bildschirm**. Es wird ein Fenster mit der Bestätigung der Bestellung geöffnet.
