# Lieferantenbeleg erfassen

So erstellen Sie einen neuen Auftrag für einen Lieferanten und erfassen die Artikelpositionen.

**Video:** [Lieferantenbeleg erfassen](https://youtu.be/_wONgD0iakI)

## Schritte

1. Wählen Sie den Menüpunkt **Auftr. Lieferanten** → **Auftrag Lieferanten** aus.
2. Klicken Sie auf das Symbol **Neuer Datensatz** in der Werkzeugleiste, um einen neuen Lieferantenauftrag zu erstellen.
3. Geben Sie die ersten Buchstaben des gewünschten Lieferanten ein (z. B. „CO“) und drücken Sie **ENTER** oder klicken Sie auf den Button **…** rechts neben dem Feld Matchcode. Es wird eine Liste mit passenden Lieferanten angezeigt. (Die Liste erscheint nur, wenn mehrere Lieferanten dem Suchkriterium entsprechen.)
4. Wählen Sie den gewünschten Lieferanten durch **ENTER** oder Doppelklick aus. Der Lieferantenbeleg ist nun diesem Lieferanten zugeordnet.
5. Speichern Sie den neu erstellten Lieferantenbeleg mit dem **Diskettensymbol** oder mit **F10**.

### Artikelpositionen hinzufügen

6. Klicken Sie auf das Symbol **Neuer Datensatz** in der Werkzeugleiste der Positionsliste. Eine neue Artikelposition wird erzeugt, das Feld **RefNr** wird angesprungen.
7. Im Feld **RefNr** geben Sie eine Lieferanten-Artikelnummer oder einen Barcode ein (oder scannen Sie den Artikel). Drücken Sie **ENTER** oder klicken Sie auf **…**. Bei mehreren Treffern wählen Sie den richtigen Artikel aus.
8. Es wird eine Preisfindung durchgeführt; der Einkaufspreis wird angezeigt. Korrigieren Sie ihn ggf. (der Einkaufspreis gilt für die in der Position angegebene Mengeneinheit, z. B. „Gebinde“).
9. Erfassen Sie im Feld **Menge** die gewünschte Menge und ggf. die Gratis-Menge.
10. Speichern Sie die Position mit dem **Diskettensymbol** oder **F10**.
11. Wiederholen Sie die Schritte 6–10 für alle weiteren Artikelpositionen.
