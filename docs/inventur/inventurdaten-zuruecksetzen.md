# Inventurdaten für Lager zurücksetzen

So setzen Sie die Inventurdaten für ein Lager zurück.

**Wichtig:** Damit wird die **aktuelle Inventur gelöscht** und die Daten zurückgesetzt. Führen Sie diesen Schritt **nur** aus, wenn die Inventur **nicht** verbucht werden soll.

## Vorgehen

- **Datenpflege** → **Inventur** → **Inventurdaten für Hauptlager zurücksetzen**
