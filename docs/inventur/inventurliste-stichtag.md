# Inventurliste per Stichtag erstellen

So erstellen Sie eine Inventurliste auf Basis eines Stichtags.

## Schritte

1. **Stammdaten** → **Artikel** → **Selektion** (Pfeil runter) → **Artikel für Inventurausdruck**.  
   Wählen Sie unter dem Menüpunkt **Artikel** die Selektion (Pfeil runter) → **Artikel für Inventurausdruck**. (Artikel werden selektiert – bitte warten.)
2. **Zusätze** → **Inventur** → **Inventurliste per Stichtag**.  
   Wählen Sie im Fenstermenü **Inventur** → **Inventurliste per Stichtag**.
3. Bei **Inventur per Stichtag** öffnet sich ein kleines Fenster: Klicken Sie auf **Daten neu berechnen** und bestätigen Sie mit **OK**. Im folgenden Fenster tragen Sie das Datum für die Inventurberechnung ein (z. B. 31.12.20…) und bestätigen mit **OK**.
