# Inventurerfassung an der Kasse

So erfassen Sie die Inventur an der Kasse.

**Videos:** [Inventur Modus ein](https://youtu.be/-n_bCE5U3DM) · [Inventur Modus aus](https://youtu.be/gKop6hGWrzQ)

## Voraussetzung

- Ein Kassierer muss angemeldet sein.

## Inventurmodus einschalten

1. Stellen Sie den Modus auf **REG2** (durch Betätigen der Taste **REG**).
2. Geben Sie **4070** ein und drücken Sie die Taste **ZWISCHENSUMME**. Die Kasse zeigt „TRAINING EIN“ oder „INVENTUR EIN“ an.

## Artikel erfassen

3. Bearbeiten Sie die Artikel wie beim Verkauf:
   - Menge eingeben, Taste **X** betätigen, Artikel scannen.
   - Nach maximal **50 Artikeln**: Taste **Zwischensumme** und Taste **BAR** (Bon abschließen).

## Inventurmodus ausschalten

4. Geben Sie **4071** ein und bestätigen Sie mit der Taste **ZWISCHENSUMME**. Die Kasse zeigt „TRAINING AUS“ oder „INVENTUR AUS“ an.
5. Nach der Inventur: Modus wieder auf **REG1** stellen.

## Nächster Schritt in MD-Premium

Etwa **10 Minuten** nach der letzten Inventur an der Kasse sind die Daten in MD-Premium übertragen. Sie können die erfassten Artikel dann unter **Datenpflege / Inventur / Inventurartikel bearbeiten** bearbeiten (siehe [Inventurdaten verbuchen](/docs/inventur/inventurdaten-verbuchen)).
