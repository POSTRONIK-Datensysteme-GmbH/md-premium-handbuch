# Artikel ohne Inventur seit Datum – Lagerstand bereinigen

So setzen Sie den Lagerstand von Artikeln auf 0, die seit einem bestimmten Datum nicht mehr inventiert wurden (nicht mehr auf Lager).

**Video:** [Lagerstand bereinigen](https://youtu.be/oAPw1HhXhxs)

## Schritte

1. **Stammdaten** → **Artikel** → **Selektion** (Pfeil runter) → **Artikel ohne Inventur seit Datum**.  
   Unter diesem Punkt werden alle Artikel ausgewählt, die noch keine Inventur haben (nicht mehr auf Lager).
2. **Stammdaten** → **Artikel** → **Zusätze** → **Inventur** → **Inventurdaten setzen**.  
   Mit diesem Schritt setzen Sie alle im vorherigen Schritt ausgewählten Artikel in den Inventurstatus.
3. **Datenpflege** → **Inventur** → **Differenzen buchen**.  
   Der Lagerstand der selektierten Artikel wird auf **0** gestellt.
