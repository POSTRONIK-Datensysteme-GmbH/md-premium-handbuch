# Bestehende Artikel – Verkaufspreis ändern

So ändern Sie den Verkaufspreis eines bestehenden Artikels.

**Video:** [Verkaufspreis ändern](https://youtu.be/yzCTto9EC4I)

## Schritte

1. Suchen Sie den Artikel wie in der Anleitung [Bestehende Artikel suchen nach Barcode](/docs/stammdaten/bestehende-artikel-suchen-barcode) (oder nach Bezeichnung/Warengruppe) beschrieben.
2. Klicken Sie auf den Reiter **(1) Preise**. Der aktuelle Preis wird angezeigt.
3. Markieren Sie den alten Preis im Feld **(2) Betrag Inkl.** und überschreiben Sie diesen.
4. Klicken Sie auf das **(3) Speichern**-Symbol oder drücken Sie die Taste **F10**, um die Änderungen zu speichern.
