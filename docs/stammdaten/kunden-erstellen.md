# Kunden erstellen

So erstellen Sie einen Kunden manuell.

## Schritte

1. Wählen Sie im Hauptmenü **Stammdaten** → **Kunden**. Das Hauptfenster **Kunden** wird geöffnet und zeigt den zuletzt geöffneten Kunden an. Detaillierte Beschreibungen finden Sie auch unter MDKND.

   ![Kundenmaske](/img/docs/stammdaten-kunden-erstellen-1.png)

2. Klicken Sie auf **Neuer Datensatz** (![Neuer Datensatz](/img/docs/fw_newbutton.png)). Das Feld **Knd Id** in der Gruppe **ID** wird automatisch mit der nächsten verfügbaren Kundennummer befüllt:

   ![Knd Id automatisch befüllt](/img/docs/stammdaten-kunden-erstellen-2.png)

3. Geben Sie im Feld **Match** (MATCHCODE) einen Suchbegriff an, der Ihnen später dabei hilft, den Kunden schnell zu finden, z. B. den Nachnamen oder Firmennamen des Kunden.

4. Wählen Sie im Feld **Adressart**, ob es sich bei dem Kunden um eine Firma, eine weibliche oder eine männliche Person handelt, oder ob die Adressart bzw. das Geschlecht unbekannt ist.

   :::info
   Die Adressart wirkt sich unter anderem auf die Anrede in Aussendungen aus.
   :::

5. Füllen Sie alle weiteren Felder in der Gruppe **ID** aus, die für Sie relevant sind.

   ![Gruppe ID](/img/docs/stammdaten-kunden-erstellen-3.png)

6. Geben Sie in der Gruppe **Adresse** die Adressdaten des Kunden an.

   ![Gruppe Adresse](/img/docs/stammdaten-kunden-erstellen-4.png)

7. Geben Sie in der Gruppe **Kontakt** die Kontaktdaten des Kunden an.

   ![Gruppe Kontakt](/img/docs/stammdaten-kunden-erstellen-5.png)

8. Bearbeiten Sie in der Gruppe **#** nach Bedarf die folgenden Felder und Optionen:
   - **Kennung:** Feld zur Angabe eines oder mehrerer vordefinierter Zeichen, die das Programmverhalten verändern. Für allgemeine Informationen siehe Kennungen. Für eine Beschreibung der verwendbaren Zeichen siehe Kennungen: Kunden.

9. Wenn Sie alle für Sie relevanten Informationen zum Kunden eingegeben haben, klicken Sie auf **Speichern** (![Speichern](/img/docs/fw_savebutton.png)).
