# Bestehende Artikel suchen nach Bezeichnung

So finden Sie einen Artikel in der Artikeldatenbank anhand seiner Bezeichnung.

**Video:** [Bestehende Artikel suchen nach Bezeichnung](https://youtu.be/mbJhea9m2yk)

## Schritte

1. Öffnen Sie die Artikelmaske: **Stammdaten** → **Artikel**.
2. Drücken Sie das **Suchen-Symbol** oder die Taste **F3**, um die Suche zu starten.
3. Geben Sie im Feld **Matchcode** die Bezeichnung des gesuchten Artikels ein. Setzen Sie am Anfang und Ende ein **%** (z. B. `%MARLB%`).
4. Das System zeigt alle Artikel an, die den eingegebenen Zeichensatz enthalten. Sie können die gefundenen Artikel mit den blauen Pfeiltasten durchblättern oder durch Klick auf **Listenansicht** (bzw. Taste **F2**) in einer Tabelle ansehen.
5. Durch Klicken auf einen Spaltenkopf sortieren Sie die Tabelle nach dieser Spalte.
6. Mit **Doppelklick** auf den Zeilenkopf öffnen Sie die Detailansicht des Artikels.
7. Die Listenansicht schließen Sie mit Klick auf das **X** oder mit der Taste **ESC**.
