# Bestehende Artikel suchen nach Warengruppe

So finden Sie alle Artikel einer bestimmten Warengruppe.

**Video:** [Bestehende Artikel suchen nach Warengruppe](https://youtu.be/jBWbNb3AjcE)

## Schritte

1. Öffnen Sie die Artikelmaske: **Stammdaten** → **Artikel**.
2. Klicken Sie auf das **Auswahlfenster-Symbol** beim Feld **WGR** und wählen Sie im neuen Fenster den Button **Alle**.
3. Wählen Sie die gewünschte Warengruppe durch **Doppelklick** auf den Zeilenkopf aus.
4. Drücken Sie das **Suchen-Symbol** oder die Taste **F3**, um die Suche zu starten.
5. Das System zeigt alle Artikel an, die der gewählten Warengruppe zugeordnet sind. Sie können die gefundenen Artikel mit den blauen Pfeiltasten durchblättern oder durch Klick auf **Listenansicht** (bzw. Taste **F2**) in einer Tabelle ansehen.
6. Durch Klicken auf einen Spaltenkopf sortieren Sie die Tabelle nach dieser Spalte.
7. Mit **Doppelklick** auf den Zeilenkopf öffnen Sie die Detailansicht des Artikels.
8. Die Listenansicht schließen Sie mit Klick auf das **X** oder mit der Taste **ESC**.
