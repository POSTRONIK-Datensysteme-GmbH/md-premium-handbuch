# Lieferanten erstellen

So erstellen Sie einen Lieferanten.

## Schritte

1. Wählen Sie im Hauptmenü **Stammdaten** → **Lieferanten**. Das Hauptfenster **Lieferanten** wird geöffnet und zeigt den zuletzt geöffneten Lieferanten an. Detaillierte Beschreibungen finden Sie auch unter MDLIE.

   ![Lieferantenmaske](/img/docs/stammdaten-lieferanten-erstellen-1.png)

2. Klicken Sie auf **Neuer Datensatz** (![Neuer Datensatz](/img/docs/fw_newbutton.png)), um einen neuen Lieferanten zu erstellen. Das Feld **Lie Id** in der Gruppe **ID** wird automatisch mit der nächsten verfügbaren Lieferantennummer befüllt:

   ![Lie Id automatisch befüllt](/img/docs/stammdaten-lieferanten-erstellen-2.png)

3. Geben Sie im Feld **Match** (MATCHCODE) einen Suchbegriff an, der Ihnen später dabei hilft, den Lieferanten schnell zu finden, z. B. den Firmennamen des Lieferanten.

4. Wählen Sie im Feld **Adressart**, ob es sich bei dem Lieferanten um eine Firma, eine weibliche oder eine männliche Person handelt, oder ob die Adressart bzw. das Geschlecht unbekannt ist.

   :::info
   Die Adressart wirkt sich unter anderem auf die Anrede in Aussendungen aus.
   :::

5. Füllen Sie alle weiteren Felder in der Gruppe **ID** aus, die für Sie relevant sind.

   ![Gruppe ID](/img/docs/stammdaten-lieferanten-erstellen-3.png)

6. Geben Sie in der Gruppe **Adresse** die Adressdaten des Lieferanten an.

   ![Gruppe Adresse](/img/docs/stammdaten-lieferanten-erstellen-4.png)

7. Geben Sie in der Gruppe **Kontakt** die Kontaktdaten des Lieferanten an.

   ![Gruppe Kontakt](/img/docs/stammdaten-lieferanten-erstellen-5.png)

8. Wenn Sie alle für Sie relevanten Informationen zum Lieferanten eingegeben haben, klicken Sie auf **Speichern** (![Speichern](/img/docs/fw_savebutton.png)).
