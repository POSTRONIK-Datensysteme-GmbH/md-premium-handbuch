# Bestehende Artikel suchen nach Barcode

So finden Sie einen Artikel in der Warenwirtschaft anhand seines Barcodes (EAN) oder der Lieferanten-Artikelnummer und öffnen das passende Stammblatt.

**Video:** Die Schritte sehen Sie auch in diesem Video: [Bestehende Artikel suchen nach Barcode](https://youtu.be/l7bOeU96nCM)

## Schritte

1. Öffnen Sie die Artikelmaske: **Stammdaten** → **Artikel**.
2. Klicken Sie auf das **Barcode-Symbol** (Barcode-Zeichen) in der oberen rechten Ecke des Fensters.
3. Geben Sie den Barcode oder die Lieferanten-Artikelnummer ein – oder scannen Sie den Artikel mit dem Scanner.
4. Drücken Sie **OK**. Der Artikel wird angezeigt.

Wird ein leeres Datenblatt angezeigt, ist der Artikel im System nicht vorhanden. Die Referenznummer ist pro Artikel eindeutig.

## Hinweis: UPC- und EAN-Codes

Bei manchen Produkten ist ein amerikanischer **UPC-Code** (12 Stellen) aufgedruckt. Das Kassensystem wandelt ihn in einen europäischen **EAN-Code** (13 Stellen) um, indem eine **0** vorangestellt wird. In der Warenwirtschaft muss der Barcode in dieser EAN-Form angelegt sein.  
Beispiel: Aus `070330600065` wird `0070330600065`.
