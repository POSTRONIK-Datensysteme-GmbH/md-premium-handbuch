# XMPS Abschlüsse

Übersicht der Abschluss-Themen der XMPS Kassensoftware. Detaillierte Anleitungen und Videos: **POStronik Support** → [Abschlüsse](https://support.postronik.at/help/de-de/15-abschlusse).

| Thema | Kurzbeschreibung | Anleitung |
|-------|------------------|-----------|
| **Wechselgelddeklaration** | Ladenanfangsbestand bei Kassierbeginn deklarieren; nach Anmelden durchführen | [Wechselgelddeklaration](https://support.postronik.at/help/de-de/15-abschlusse/47-wechselgelddeklaration) · [Video](https://youtu.be/7mZSyFPkba8) |
| **Geldabschöpfung** | Geldentnahmen (Bankgeld) deklarieren; nach jeder Entnahme, wird bis Tagesabschluss aufsummiert | [Geldabschöpfung](https://support.postronik.at/help/de-de/15-abschlusse/48-geldabschopfung) · [Video](https://youtu.be/DibTxGMyNR8) |
| **Geldzählung** | Aktuellen Ladeninhalt deklarieren; beliebig oft, z. B. nach Kassierende oder Kassensturz | [Geldzählung](https://support.postronik.at/help/de-de/15-abschlusse/49-geldzahlung) · [Video](https://youtu.be/F2qFLIu8zEQ) |
| **Nullbeleg** | Seite „FUNKT“ → X/Z → JA (Modus X1) → 9811 [JA]; z. B. für Finanzbeamten | [Nullbeleg](https://support.postronik.at/help/de-de/15-abschlusse/1-nullbeleg) · [Video](https://youtu.be/G5M6xz0H0A4) |
| **Jahresbeleg** | Seite „FUNKT“ → X/Z → JA (Modus X1) → 9833 [JA]; bei FinanzOnline melden | [Jahresbeleg](https://support.postronik.at/help/de-de/15-abschlusse/3-jahresbeleg) · [Video](https://youtu.be/DDohE6W_aVE) |
| **Bankomat-Tagesabschluss** | Täglich für B-K-Terminal nötig (evtl. TES-Karte); Ebene Funktion → X/Z → Z → JA → 9898 → Zwischensumme | [Bankomat-Tagesabschluss](https://support.postronik.at/help/de-de/15-abschlusse/69-bankomat-tagesabschluss) |
