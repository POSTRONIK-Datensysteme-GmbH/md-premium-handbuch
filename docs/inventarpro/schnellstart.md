# InventarPro – Schnellstart-Anleitung

Diese Anleitung hilft Ihnen bei den ersten Schritten mit **InventarPro** auf dem MDE-Gerät.

---

## 1. Vorbereitung (einmalig bei erster Inbetriebnahme)

- **WLAN verbinden:** Stellen Sie sicher, dass das MDE-Gerät mit Ihrem Firmen-WLAN verbunden ist, bevor Sie die App starten.
- **Erster Start:** Öffnen Sie die InventarPro-App. Sie sollte sich automatisch anmelden und das Hauptmenü anzeigen.
- **Datenbank erstmalig laden:**
  - Tippen Sie im Hauptmenü auf die Schaltfläche **Importieren**.
  - Warten Sie, bis der Import abgeschlossen ist. **Dieser Schritt ist entscheidend!**

---

## 2. Tägliche Routine vor Arbeitsbeginn

- **Datenbank aktualisieren:** Tippen Sie jeden Tag vor der ersten Nutzung im Hauptmenü auf **Importieren**, um sicherzustellen, dass Sie die aktuellsten Artikeldaten, Preise und Lieferanteninformationen haben.

---

## 3. Hauptmenü verstehen

- **Kacheln:** Jede Kachel startet eine Aufgabe (z. B. Lieferschein, Inventur).
- **Schaltflächen unten:**
  - **Importieren:** Lädt Daten (siehe Schritt 1 & 2).
  - **Artikel Info (i):** Zeigt Details zu einem gescannten Artikel.
  - **Exportieren:** Sendet fertige Belege an Ihr System (blinkt, wenn Daten bereitstehen).

---

## 4. Beispiel: Einen Lieferschein erfassen (Wareneingang)

1. **Modul starten:** Tippen Sie im Hauptmenü auf die Kachel **Lieferschein**.
2. **Kopfdaten eingeben:**
   - Wählen Sie den Lieferanten aus.
   - Lieferscheinnummer und Datum können Sie bei Bedarf anpassen (optional).
   - Tippen Sie auf **Neu**.
3. **Artikel scannen:**
   - Scannen Sie den Barcode des ersten Artikels.
   - Scannen Sie jeden weiteren Artikel einzeln oder passen Sie die Menge im Mengenfeld an.
4. **Beleg speichern:** Wenn alle Artikel gescannt sind, drücken Sie die **Zurück-Taste** am Gerät. Der Lieferschein erscheint in der Liste.
5. **Beleg abschließen:** Wählen Sie den Lieferschein in der Liste aus und tippen Sie auf **Abschließen**.
6. **Daten exportieren:** Gehen Sie zurück zum Hauptmenü. Wenn die Schaltfläche **Exportieren** blinkt, tippen Sie darauf, um die Daten zu senden.

---

## 5. App-Updates

- Die App prüft beim Start auf Updates. Wenn ein **„Download“-Button** erscheint, tippen Sie darauf und folgen Sie den Installationsanweisungen.

---

## Bei Problemen

Prüfen Sie zuerst die **WLAN-Verbindung**. Ansonsten kontaktieren Sie den Support.
