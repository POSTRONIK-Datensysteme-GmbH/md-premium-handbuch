# Artikli bez inventure od datuma – Čišćenje zaliha

Kako postaviti zalihu na 0 za artikle koji nisu bili u inventuri od datuma (više nisu na zalihi).

**Video:** [Čišćenje zaliha](https://youtu.be/oAPw1HhXhxs)

## Koraci

1. **Osnovni podaci** → **Artikli** → **Izbor** (strelica dole) → **Artikli bez inventure od datuma**.  
   Time se izabiru svi artikli koji nemaju inventuru (više nisu na zalihi).
2. **Osnovni podaci** → **Artikli** → **Dodatno** → **Inventar** → **Postavi podatke inventure**.  
   Time se svi artikli izabrani u prethodnom koraku postavljaju u status inventure.
3. **Održavanje podataka** → **Inventar** → **Evidentiraj razlike**.  
   Zaliha izabranih artikala postavlja se na **0**.
