# Narudžba dobavljača

Kako kreirati narudžbu dobavljača i štampati je.

**Video:** [Narudžba dobavljača](https://youtu.be/T7lyD6STjRw)

## Koraci

1. Unesite dokument dobavljača kao u [Unos dokumenta dobavljača](/docs/auftrag-lieferanten/lieferantenbeleg-erfassen).
2. Kliknite **[1] Štampa dokumenta** i izaberite **[2] Narudžba**.
3. U dijalogu možete uneti datum narudžbe.
4. Potvrdite sa **Ekran**. Otvara se prozor sa potvrdom narudžbe.
