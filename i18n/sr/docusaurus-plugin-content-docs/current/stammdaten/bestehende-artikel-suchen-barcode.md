# Pretraga postojećih artikala po barkodu

Kako pronaći artikal u sistemu po barkodu (EAN) ili broju artikla kod dobavljača i otvoriti odgovarajući osnovni zapis.

**Video:** [Pretraga postojećih artikala po barkodu](https://youtu.be/l7bOeU96nCM)

## Koraci

1. Otvorite formu za artikal: **Osnovni podaci** → **Artikli**.
2. Kliknite na **ikonu barkoda** u gornjem desnom uglu prozora.
3. Unesite barkod ili broj artikla kod dobavljača, ili skenirajte artikal skenerom.
4. Pritisnite **OK**. Artikal se prikazuje.

Ako se prikaže prazan zapis, artikal nije u sistemu. Referentni broj je jedinstven po artiklu.

## Napomena: UPC i EAN kodovi

Neki proizvodi imaju američki **UPC kod** (12 cifara). Blagajna ga pretvara u evropski **EAN kod** (13 cifara) dodavanjem vodeće **0**. Barkod mora biti sačuvan u ovom EAN obliku u sistemu. Primer: `070330600065` postaje `0070330600065`.
