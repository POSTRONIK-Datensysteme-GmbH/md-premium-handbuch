# XMPS rad na kasi

Pregled tema rada na XMPS kasi. Svi koraci su opisani u ovom priručniku.

| Tema | Kratak opis |
|------|-------------|
| **Prijava / Odjava** | Korišćenje kase kao blagajnik |
| **Množenje u prodaji** | Količina → taster „X“ → skeniranje ili PLU |
| **Ručni unos (PLU)** | Unos PLU ili EAN → taster PLU/EAN |
| **Storno** | Taster Storno → ponovo uneti artikal (samo poslednji) |
| **Prekid računa** | Prekinuti transakciju; svi artikli stornirani |
| **Uplata / Isplata** | „FUNKTION“ → iznos → Uplata ili Isplata |
| **Otvaranje ladice / Bez prodaje** | Otvoriti ladicu bez unosa |
| **Vraćanje mreže kasa** | Režim „X“ → 4301 → Međuzbir → izveštaj → REG 1 |
| **Povrati** | Povrat → skeniranje/EAN → Gotovina |
| **Popust (iznos)** | Artikal/PLU ili Međuzbir → iznos popusta → Popust iznos |
| **Popust (%)** | Artikal/PLU ili Međuzbir → % → Popust |
| **Parkiranje kupca** | „Parkiraj kupca“ / „Nastavi kupca“ |
| **Kreditna prodaja** | Broj kupca → Kupac → skeniranje → Međuzbir → Kredit |
| **Uplate kupaca (AKONTO)** | Broj kupca → Akonto → iznos → Gotovina → ESC |
| **Račun kupca + / –** | Broj kupca → Račun – → iznos → Račun – |
