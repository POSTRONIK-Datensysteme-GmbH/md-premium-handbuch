# XMPS rad na kasi

Pregled tema rada na XMPS kasi. Detaljna uputstva i video: **POStronik podrška** → [Rad](https://support.postronik.at/help/de-de/14-bedienung).

| Tema | Kratak opis | Uputstvo |
|------|-------------|----------|
| **Prijava / Odjava** | Korišćenje kase kao blagajnik | [Rad (pregled)](https://support.postronik.at/help/de-de/14-bedienung) |
| **Množenje u prodaji** | Više istih artikala: unos količine → taster „X“ → skeniranje ili PLU | [Množenje](https://support.postronik.at/help/de-de/14-bedienung/54-multiplikation-bei-verkaufsvorgangen) · [Video](https://youtu.be/JbhuffXc7ss) |
| **Ručni unos (PLU)** | Unos PLU ili EAN → taster PLU/EAN | [Taster PLU](https://support.postronik.at/help/de-de/14-bedienung/52-manuelle-verbuchung-von-artikeln-uber-plu-taste) · [Video](https://youtu.be/nz-v3yCnvZA) |
| **Storno** | Taster Storno → ponovo uneti artikal za storno (samo poslednji) | [Storno](https://support.postronik.at/help/de-de/14-bedienung/57-storno) · [Video](https://youtu.be/WCb3dXabpJM) |
| **Prekid računa** | Prekinuti transakciju; svi artikli stornirani; npr. pri više grešaka | [Prekid računa](https://support.postronik.at/help/de-de/14-bedienung/58-bon-abbruch) · [Video](https://youtu.be/UGTz2CSr82E) |
| **Uplata / Isplata** | Ekran „FUNKTION“ → iznos → Uplata ili Isplata (npr. pouzećem) | [Uplata/Isplata](https://support.postronik.at/help/de-de/14-bedienung/55-aus-oder-einzahlung-buchen) · [Video](https://youtu.be/syl1nxaN7A0) |
| **Otvaranje ladice / Bez prodaje** | Otvoriti ladicu bez unosa (npr. kusur) | [Otvaranje ladice](https://support.postronik.at/help/de-de/14-bedienung/56-lade-offnen-kein-verkauf) |
| **Vraćanje mreže kasa** | Posle mrežnog problema: Režim „X“ → 4301 → Međuzbir → štampa izveštaj → REG 1 | [Mreža kasa](https://support.postronik.at/help/de-de/14-bedienung/24-kassenverbund-wiederherstellen) |
| **Povrati** | Povrat → skeniranje/EAN → zatvoriti Gotovinom (korekcija zaliha i zbira) | [Povrati](https://support.postronik.at/help/de-de/14-bedienung/59-artikel-retouren) |
| **Popust (iznos)** | Artikal/PLU ili Međuzbir → iznos popusta → taster Popust iznos | [Popust iznos](https://support.postronik.at/help/de-de/14-bedienung/63-nachlass-betrag) |
| **Popust (%)** | Artikal/PLU ili Međuzbir → % → taster Popust (može biti unapred podešen) | [Popust %](https://support.postronik.at/help/de-de/14-bedienung/64-rabatt) |
| **Parkiranje kupca** | Pauzirati transakciju, uslužiti drugog → „Parkiraj kupca“ / „Nastavi kupca“ | [Parkiranje kupca](https://support.postronik.at/help/de-de/14-bedienung/65-kunde-parken) |
| **Kreditna prodaja** | Broj kupca → Kupac → skeniranje → Međuzbir → Kredit | [Kreditna prodaja](https://support.postronik.at/help/de-de/14-bedienung/66-krediteinkaufe-mit-kundennummer) |
| **Uplate kupaca (AKONTO)** | Broj kupca → Akonto → iznos → Gotovina → ESC | [AKONTO](https://support.postronik.at/help/de-de/14-bedienung/67-kundeneinzahlungen-mit-akonto) |
| **Račun kupca + / –** | Promena salda bez gotovine (npr. bankarski transfer): Broj kupca → Račun – → iznos → Račun – | [Račun kupca +/–](https://support.postronik.at/help/de-de/14-bedienung/68-kundenkonto-kundenkonto) |
