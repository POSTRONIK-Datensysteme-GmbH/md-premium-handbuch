# XMPS zatvaranja

Pregled tema zatvaranja XMPS kase. Detaljna uputstva i video: **POStronik podrška** → [Zatvaranja](https://support.postronik.at/help/de-de/15-abschlusse).

| Tema | Kratak opis | Uputstvo |
|------|-------------|----------|
| **Deklaracija početnog stanja** | Deklarisati početno stanje na početku smene; posle prijave | [Početno stanje](https://support.postronik.at/help/de-de/15-abschlusse/47-wechselgelddeklaration) · [Video](https://youtu.be/7mZSyFPkba8) |
| **Odliv gotovine** | Deklarisati podizanja (depozit u banku); sumira se do kraja dana | [Odliv](https://support.postronik.at/help/de-de/15-abschlusse/48-geldabschopfung) · [Video](https://youtu.be/DibTxGMyNR8) |
| **Prebrojavanje** | Deklarisati trenutni sadržaj ladice; bilo kada, npr. kraj smene | [Prebrojavanje](https://support.postronik.at/help/de-de/15-abschlusse/49-geldzahlung) · [Video](https://youtu.be/F2qFLIu8zEQ) |
| **Null račun** | Ekran „FUNKT“ → X/Z → DA (Režim X1) → 9811 [DA]; npr. za kontrolu | [Null račun](https://support.postronik.at/help/de-de/15-abschlusse/1-nullbeleg) · [Video](https://youtu.be/G5M6xz0H0A4) |
| **Godišnji račun** | Ekran „FUNKT“ → X/Z → DA (Režim X1) → 9833 [DA]; prijava u FinanzOnline | [Godišnji račun](https://support.postronik.at/help/de-de/15-abschlusse/3-jahresbeleg) · [Video](https://youtu.be/DDohE6W_aVE) |
| **Dnevno zatvaranje bankomata** | Dnevno obavezno za terminal (event. TES kartica); Funkcija → X/Z → Z → DA → 9898 → Međuzbir | [Zatvaranje bankomata](https://support.postronik.at/help/de-de/15-abschlusse/69-bankomat-tagesabschluss) |
