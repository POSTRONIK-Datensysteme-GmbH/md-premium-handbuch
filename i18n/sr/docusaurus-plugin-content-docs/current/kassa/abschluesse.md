# XMPS zatvaranja

Pregled tema zatvaranja XMPS kase. Svi koraci su opisani u ovom priručniku.

| Tema | Kratak opis |
|------|-------------|
| **Deklaracija početnog stanja** | Na početku smene; posle prijave |
| **Odliv gotovine** | Podizanja; sumira se do kraja dana |
| **Prebrojavanje** | Trenutni sadržaj ladice; npr. kraj smene |
| **Null račun** | „FUNKT“ → X/Z → DA (X1) → 9811 [DA] |
| **Godišnji račun** | „FUNKT“ → X/Z → DA (X1) → 9833 [DA]; FinanzOnline |
| **Dnevno zatvaranje bankomata** | Funkcija → X/Z → Z → DA → 9898 → Međuzbir |
