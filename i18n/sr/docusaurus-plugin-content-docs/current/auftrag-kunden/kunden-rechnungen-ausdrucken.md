# Štampanje računa kupaca

Kako štampati račune kupaca (kombinovani blagajnički račun) ili ih poslati e-poštom.

## Koraci

1. U meniju idite na **Narudžbe kupaca** -> **Narudžbe kupaca** za prikaz poslednjeg uređenog dokumenta kupca.
2. Izaberite **Izbor** -> **KA dokumenti** i filtrirajte po **ID kupca** i **Datum**.
3. Unesite **ID kupca** (6 cifara) i **raspon datuma** i potvrdite sa **OK**.

**Napomena ID kupca:** Ako je kupac kreditiran na blagajni 21, ID kupca je **200021**; za blagajnu 101 je **200101**.

4. Izaberite **Štampa dokumenta** -> **Kombinovana štampa dokumenta** -> **Kombinovani blagajnički račun**.
5. Štampajte kombinovani račun ili ga pošaljite e-poštom (ako je adresa e-pošte sačuvana) klikom na **Eksport**. (Sa Outlook-om se otvara e-pošta.)
