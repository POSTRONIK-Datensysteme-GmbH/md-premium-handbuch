# Buscar artículos por grupo de productos

Cómo encontrar todos los artículos de un grupo de productos.

**Vídeo:** [Buscar artículos por grupo de productos](https://youtu.be/jBWbNb3AjcE)

## Pasos

1. Abra el formulario de artículos: **Datos maestros** → **Artículos**.
2. Haga clic en el **icono de ventana de selección** junto a **WGR** y elija **Todos** en la nueva ventana.
3. Seleccione el grupo de productos con **doble clic** en su fila.
4. Pulse el **icono de búsqueda** o **F3** para iniciar la búsqueda.
5. El sistema mostrará todos los artículos del grupo. Use las flechas o **Vista de lista** (**F2**) para navegar.
6. Haga clic en un encabezado de columna para ordenar.
7. **Doble clic** en una fila para abrir el detalle del artículo.
8. Cierre la lista con **X** o **ESC**.
