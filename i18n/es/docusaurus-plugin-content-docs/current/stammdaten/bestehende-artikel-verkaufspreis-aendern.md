# Cambiar precio de venta de un artículo

Cómo cambiar el precio de venta de un artículo existente.

**Vídeo:** [Cambiar precio de venta](https://youtu.be/yzCTto9EC4I)

## Pasos

1. Busque el artículo como en [Buscar artículos por código de barras](/docs/stammdaten/bestehende-artikel-suchen-barcode) (o por descripción/grupo).
2. Abra la pestaña **(1) Precios**. Se muestra el precio actual.
3. Seleccione el precio anterior en **(2) Importe incl.** y sobrescríbalo.
4. Haga clic en el icono **(3) Guardar** o pulse **F10**.
