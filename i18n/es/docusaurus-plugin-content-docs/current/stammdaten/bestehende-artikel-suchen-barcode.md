# Buscar artículos por código de barras

Cómo encontrar un artículo en el sistema por su código de barras (EAN) o número de artículo del proveedor y abrir el registro maestro correspondiente.

**Vídeo:** [Buscar artículos por código de barras](https://youtu.be/l7bOeU96nCM)

## Pasos

1. Abra el formulario de artículos: **Datos maestros** → **Artículos**.
2. Haga clic en el **icono de código de barras** en la esquina superior derecha de la ventana.
3. Introduzca el código de barras o el número de artículo del proveedor, o escanee el artículo.
4. Pulse **OK**. Se mostrará el artículo.

Si aparece un registro vacío, el artículo no está en el sistema. El número de referencia es único por artículo.

## Nota: códigos UPC y EAN

Algunos productos tienen un **código UPC** estadounidense (12 dígitos). El sistema de caja lo convierte en **código EAN** europeo (13 dígitos) añadiendo un **0** al inicio. El código de barras debe guardarse en formato EAN en el sistema. Ejemplo: `070330600065` se convierte en `0070330600065`.
