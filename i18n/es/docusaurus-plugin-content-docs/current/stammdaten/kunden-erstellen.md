# Crear cliente

Cómo crear un cliente manualmente.

## Pasos

1. En el menú principal, seleccione **Datos maestros** → **Clientes**. Se abre la ventana **Clientes** y muestra el último cliente abierto. Para descripciones detalladas consulte también MDKND.

   ![Formulario de cliente](/img/docs/stammdaten-kunden-erstellen-1.png)

2. Haga clic en **Nuevo registro** (![Nuevo registro](/img/docs/fw_newbutton.png)). El campo **Knd Id** del grupo **ID** se rellena automáticamente con el siguiente número de cliente disponible:

   ![Knd Id rellenado automáticamente](/img/docs/stammdaten-kunden-erstellen-2.png)

3. En el campo **Match** (MATCHCODE), introduzca un término de búsqueda que le ayude a encontrar el cliente rápidamente más tarde, p. ej. el apellido o nombre de la empresa del cliente.

4. En el campo **Adressart** (tipo de dirección), elija si el cliente es una empresa, una persona femenina o masculina, o si el tipo de dirección o el género es desconocido.

   :::info
   El tipo de dirección influye, entre otras cosas, en el saludo en las comunicaciones enviadas.
   :::

5. Rellene todos los demás campos del grupo **ID** que sean relevantes para usted.

   ![Grupo ID](/img/docs/stammdaten-kunden-erstellen-3.png)

6. Introduzca los datos de dirección del cliente en el grupo **Adresse** (Dirección).

   ![Grupo Dirección](/img/docs/stammdaten-kunden-erstellen-4.png)

7. Introduzca los datos de contacto del cliente en el grupo **Kontakt** (Contacto).

   ![Grupo Contacto](/img/docs/stammdaten-kunden-erstellen-5.png)

8. En el grupo **#**, edite según necesidad los siguientes campos y opciones:
   - **Kennung:** Campo para uno o más caracteres predefinidos que modifican el comportamiento del programa. Para información general véase Kennungen. Para la descripción de los caracteres utilizables véase Kennungen: Kunden.

9. Cuando haya introducido toda la información relevante del cliente, haga clic en **Guardar** (![Guardar](/img/docs/fw_savebutton.png)).
