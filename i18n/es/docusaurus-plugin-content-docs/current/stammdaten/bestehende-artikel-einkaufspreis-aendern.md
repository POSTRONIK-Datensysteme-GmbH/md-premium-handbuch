# Cambiar precio de compra de un artículo

Cómo cambiar el precio de compra de un artículo existente.

**Vídeo:** [Cambiar precio de compra](https://youtu.be/sNBcAntVThI)

## Pasos

1. Busque el artículo como en [Buscar artículos por código de barras](/docs/stammdaten/bestehende-artikel-suchen-barcode) (o por descripción/grupo).
2. Abra la pestaña **(1) Compra**. Se muestra el precio de compra actual.
3. Seleccione el precio anterior en **(2) Importe** (precio de compra neto) y sobrescríbalo.
4. Haga clic en el icono **(3) Guardar** o pulse **F10**.
