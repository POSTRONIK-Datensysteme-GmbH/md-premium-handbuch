# Crear proveedor

Cómo crear un proveedor.

## Pasos

1. En el menú principal, seleccione **Datos maestros** → **Proveedores**. Se abre la ventana **Proveedores** y muestra el último proveedor abierto. Para descripciones detalladas consulte también MDLIE.

   ![Formulario de proveedor](/img/docs/stammdaten-lieferanten-erstellen-1.png)

2. Haga clic en **Nuevo registro** (![Nuevo registro](/img/docs/fw_newbutton.png)) para crear un nuevo proveedor. El campo **Lie Id** del grupo **ID** se rellena automáticamente con el siguiente número de proveedor disponible:

   ![Lie Id rellenado automáticamente](/img/docs/stammdaten-lieferanten-erstellen-2.png)

3. En el campo **Match** (MATCHCODE), introduzca un término de búsqueda que le ayude a encontrar el proveedor rápidamente más tarde, p. ej. el nombre de la empresa del proveedor.

4. En el campo **Adressart** (tipo de dirección), elija si el proveedor es una empresa, una persona femenina o masculina, o si el tipo de dirección o el género es desconocido.

   :::info
   El tipo de dirección influye, entre otras cosas, en el saludo en las comunicaciones enviadas.
   :::

5. Rellene todos los demás campos del grupo **ID** que sean relevantes para usted.

   ![Grupo ID](/img/docs/stammdaten-lieferanten-erstellen-3.png)

6. Introduzca los datos de dirección del proveedor en el grupo **Adresse** (Dirección).

   ![Grupo Dirección](/img/docs/stammdaten-lieferanten-erstellen-4.png)

7. Introduzca los datos de contacto del proveedor en el grupo **Kontakt** (Contacto).

   ![Grupo Contacto](/img/docs/stammdaten-lieferanten-erstellen-5.png)

8. Cuando haya introducido toda la información relevante del proveedor, haga clic en **Guardar** (![Guardar](/img/docs/fw_savebutton.png)).
