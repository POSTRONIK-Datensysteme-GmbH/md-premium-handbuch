# Buscar artículos por descripción

Cómo encontrar un artículo en la base de datos por su descripción.

**Vídeo:** [Buscar artículos por descripción](https://youtu.be/mbJhea9m2yk)

## Pasos

1. Abra el formulario de artículos: **Datos maestros** -> **Artículos**.
2. Pulse el **icono de búsqueda** o la tecla **F3** para iniciar la búsqueda.
3. En el campo **Matchcode**, introduzca la descripción con **%** al inicio y al final (ej. `%MARLB%`).
4. El sistema mostrará todos los artículos que contengan ese texto. Use las flechas o **Vista de lista** (**F2**).
5. Haga clic en un encabezado de columna para ordenar.
6. **Doble clic** en el encabezado de fila para abrir el detalle.
7. Cierre la lista con **X** o **ESC**.
