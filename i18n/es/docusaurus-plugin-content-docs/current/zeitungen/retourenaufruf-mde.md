# Enviar llamada de devolución al dispositivo portátil

Cómo enviar una llamada de devolución al dispositivo portátil.

## Steps

1. Vaya a **Newspapers** -> **New return calls**.
2. **Clic derecho** en la fila de devolución y elija **Set MDE flag** en el menú contextual.
3. Cuando la transferencia al dispositivo portátil se realiza correctamente, se escribe una **marca de tiempo** en el campo **Date**.
