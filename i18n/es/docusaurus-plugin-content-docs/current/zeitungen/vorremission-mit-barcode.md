# Predevolución con código de barras

Cómo añadir artículos por código de barras/EAN a una llamada de devolución.

## Prerequisite

Debe existir una **llamada de devolución abierta** para el proveedor.

## Steps

1. Haga clic en **Newspapers** → **Pre-return with barcode**.
2. Introduzca el EAN en **[1] RefNr** o escanee con el lector de códigos de barras. Confirme con **[2] Display**.
3. Para añadir el artículo a una llamada de devolución:
   - **Clic derecho** en la fila del artículo → **Add to return order…**, o
   - **Doble clic** en el encabezado de fila del artículo.

La cantidad introducida se añade al pedido de devolución **más reciente** de ese proveedor.

**Notas:** Al añadir, la "fecha de comisión" se establece en hoy. Un artículo no puede añadirse si ya está en la llamada de devolución con el mismo número de ejemplar.
