# Devoluciones

Después de importar las llamadas de devolución desde datos EH-2000, se crean como tipo de documento **AM** (llamada de devolución). Las llamadas de devolución pueden recalcularse (cantidades establecidas según stock) e imprimirse tantas veces como sea necesario. Los artículos de periódicos pueden añadirse a llamadas de devolución existentes para predevolución – manualmente o desde artículos entregados (véase [Pre-return from delivery notes](/docs/zeitungen/vorremission-aus-lieferscheinen), [Pre-return with barcode](/docs/zeitungen/vorremission-mit-barcode)). Después de registrar todas las cantidades de devolución, las llamadas de devolución se convierten mediante impresión de documento en **devolución** (código de pedido **ZM**). Los documentos de devolución se envían al mayorista automáticamente.

## Open and print return call

1. Vaya a **Newspapers** -> **New return calls**. La ventana muestra todas las llamadas de devolución de periódicos con totales netos.
2. Abra una llamada de devolución con **doble clic** en el encabezado de fila. Asegúrese de que la pestaña **Articles** esté seleccionada.
3. Imprima mediante el menú de ventana **Document print** -> **Return order (AM)**.
4. Use la impresión para localizar los artículos de periódicos en la zona de ventas y almacén. Introduzca el **recuento real encontrado** en el documento. El documento puede cerrarse y reabrirse en cualquier momento.
