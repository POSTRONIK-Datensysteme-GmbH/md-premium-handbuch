# Imprimir facturas de clientes

Cómo imprimir facturas de clientes (ticket de caja combinado) o enviarlas por correo electrónico.

## Steps

1. En el menú vaya a **Customer orders** -> **Order customers** para mostrar el último documento de cliente editado.
2. Elija **Selection** -> **KA documents** y filtre por **Customer ID** y **Date**.
3. Introduzca la **Customer ID** (6 dígitos) y el **rango de fechas** y confirme con **OK**.

**Nota Customer ID:** Si el cliente tiene crédito en la caja 21, la Customer ID es **200021**; para la caja 101 es **200101**.

4. Elija **Document print** -> **Combined document print** -> **Combined till receipt**.
5. Imprima el ticket combinado o envíelo por correo electrónico (si hay una dirección de correo almacenada) haciendo clic en **Export**. (Con Outlook se abre un correo.)
