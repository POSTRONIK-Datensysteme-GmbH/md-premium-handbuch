# Pedido de proveedor a partir del sugerido

Cómo ejecutar una sugerencia de pedido y realizar el pedido.

**Video:** [Supplier order from order suggestion](https://youtu.be/tq-oQN8HqFE)

## Steps

1. En la barra de menú haga clic en **Execute order suggestion**.
2. En la nueva ventana seleccione el proveedor y confirme con **OK**.

Se calcula la sugerencia. El pie muestra el número de artículos y el coste total de compra.

### Ajustar cantidades y realizar pedido

3. Para cambiar las cantidades sugeridas: haga clic en el campo **Quantity** e introduzca la cantidad deseada.
4. Para realizar el pedido: en el menú de ventana abra **Extras** y elija **Order creation**. Confirme con **Yes**, espere el proceso y luego confirme el mensaje de finalización. El programa muestra el pedido.

## Columns in the order suggestion

- **WGR:** Grupo de producto del artículo
- **Art_ID:** Número interno del artículo (también número de pedido)
- **Match:** Descripción del artículo
- **Menge:** Cantidad del pedido
- **Meh:** Unidad (a menudo caja)
- **LagerMe:** Stock actual en unidad de pedido
- **LagMEH:** Stock actual en unidad base (pack)
- **MinBest:** Stock mínimo
- **Ekp:** Precio de compra por unidad de pedido
- **EKP-Summe / Vkp.Inkl-Sum:** Totales
- **MengeVK7TA:** Packs vendidos en los últimos 7 días

## Special suppliers

- **Tobaccoland:** Menú de ventana **Extras** → **Tobaccoland Order 2019** (véase [Tobaccoland – Send order online](/docs/auftrag-lieferanten/tobaccoland-bestellung-online)).
- **Philip Morris:** **Extras** → **Philip Morris Order** (véase [Philip Morris – Send order online](/docs/auftrag-lieferanten/philip-morris-bestellung-online)).
- **Moosmayr:** **Extras** → **Moosmayr Order Online** (véase [Moosmayr – Send order online](/docs/auftrag-lieferanten/moosmayr-bestellung-online)).
- **Otros proveedores de tabaco (p. ej. Mossmayr, Danzcek):** Envíe la sugerencia de pedido por correo electrónico.
