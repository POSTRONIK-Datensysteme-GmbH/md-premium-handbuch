# Moosmayr – Enviar pedido en línea

Cómo enviar un pedido a Moosmayr.

## Steps

1. En el menú **Supplier orders** seleccione un pedido de Moosmayr.
2. En **Extras** elija **Moosmayr Order Online**.
3. Aparece una ventana emergente con el número de pedido.

**Variante 1:** El pedido se envió correctamente.

**Variante 2:** Si el mensaje después de "|live|" muestra un **valor negativo** con un número de pedido, ese pedido no pudo enviarse. El número de pedido mostrado no puede solicitarse a Moosmayr – **esta línea debe eliminarse del pedido** antes de poder enviarlo.
