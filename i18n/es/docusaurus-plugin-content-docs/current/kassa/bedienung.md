# Uso de XMPS

Resumen de los temas de uso del software de caja XMPS. Guías y vídeos detallados: **Soporte POStronik** → [Uso](https://support.postronik.at/help/de-de/14-bedienung).

| Tema | Descripción breve | Guía |
|------|-------------------|------|
| **Identificación / Cierre de sesión** | Uso de caja como cajero | [Uso (resumen)](https://support.postronik.at/help/de-de/14-bedienung) |
| **Cantidad en ventas** | Varios artículos iguales: introducir cantidad → tecla „X“ → escanear o PLU | [Cantidad](https://support.postronik.at/help/de-de/14-bedienung/54-multiplikation-bei-verkaufsvorgangen) · [Vídeo](https://youtu.be/JbhuffXc7ss) |
| **Registro manual (PLU)** | Introducir PLU o EAN → tecla PLU/EAN | [Tecla PLU](https://support.postronik.at/help/de-de/14-bedienung/52-manuelle-verbuchung-von-artikeln-uber-plu-taste) · [Vídeo](https://youtu.be/nz-v3yCnvZA) |
| **Anulación** | Tecla Anulación → volver a registrar artículo a anular (solo último) | [Anulación](https://support.postronik.at/help/de-de/14-bedienung/57-storno) · [Vídeo](https://youtu.be/WCb3dXabpJM) |
| **Cancelar ticket** | Cancelar venta; todos los artículos anulados; p. ej. tras varios errores | [Cancelar ticket](https://support.postronik.at/help/de-de/14-bedienung/58-bon-abbruch) · [Vídeo](https://youtu.be/UGTz2CSr82E) |
| **Ingreso / Reintegro** | Pantalla „FUNKTION“ → importe → Ingreso o Reintegro (p. ej. contrareembolso) | [Ingreso/Reintegro](https://support.postronik.at/help/de-de/14-bedienung/55-aus-oder-einzahlung-buchen) · [Vídeo](https://youtu.be/syl1nxaN7A0) |
| **Abrir cajón / Sin venta** | Abrir cajón sin registrar (p. ej. cambio) | [Abrir cajón](https://support.postronik.at/help/de-de/14-bedienung/56-lade-offnen-kein-verkauf) |
| **Restaurar red de cajas** | Tras fallo de red: Modo „X“ → 4301 → Subtotal → imprime informe → REG 1 | [Red de cajas](https://support.postronik.at/help/de-de/14-bedienung/24-kassenverbund-wiederherstellen) |
| **Devoluciones** | Devolución → escanear/EAN → cerrar con Efectivo (corrección stock y totales) | [Devoluciones](https://support.postronik.at/help/de-de/14-bedienung/59-artikel-retouren) |
| **Descuento (importe)** | Artículo/PLU o Subtotal → importe descuento → tecla Descuento importe | [Descuento importe](https://support.postronik.at/help/de-de/14-bedienung/63-nachlass-betrag) |
| **Descuento (%)** | Artículo/PLU o Subtotal → % → tecla Descuento (puede estar prefijado) | [Descuento %](https://support.postronik.at/help/de-de/14-bedienung/64-rabatt) |
| **Cliente en espera** | Pausar venta, atender otro → „Cliente en espera“ / „Continuar cliente“ | [Cliente en espera](https://support.postronik.at/help/de-de/14-bedienung/65-kunde-parken) |
| **Ventas a crédito** | N.º cliente → Cliente → escanear → Subtotal → Crédito | [Ventas a crédito](https://support.postronik.at/help/de-de/14-bedienung/66-krediteinkaufe-mit-kundennummer) |
| **Pagos cliente (AKONTO)** | N.º cliente → Akonto → importe → Efectivo → ESC | [AKONTO](https://support.postronik.at/help/de-de/14-bedienung/67-kundeneinzahlungen-mit-akonto) |
| **Cuenta cliente + / –** | Cambiar saldo sin efectivo (p. ej. transferencia): N.º cliente → Cuenta – → importe → Cuenta – | [Cuenta cliente +/–](https://support.postronik.at/help/de-de/14-bedienung/68-kundenkonto-kundenkonto) |
