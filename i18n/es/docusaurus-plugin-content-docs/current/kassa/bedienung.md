# Uso de XMPS

Resumen de los temas de uso del software de caja XMPS. Todos los pasos se describen en este manual.

| Tema | Descripción breve |
|------|-------------------|
| **Identificación / Cierre de sesión** | Uso de caja como cajero |
| **Cantidad en ventas** | Varios artículos iguales: cantidad → tecla „X“ → escanear o PLU |
| **Registro manual (PLU)** | Introducir PLU o EAN → tecla PLU/EAN |
| **Anulación** | Tecla Anulación → volver a registrar artículo a anular (solo último) |
| **Cancelar ticket** | Cancelar venta; todos los artículos anulados |
| **Ingreso / Reintegro** | Pantalla „FUNKTION“ → importe → Ingreso o Reintegro |
| **Abrir cajón / Sin venta** | Abrir cajón sin registrar (p. ej. cambio) |
| **Restaurar red de cajas** | Modo „X“ → 4301 → Subtotal → informe → REG 1 |
| **Devoluciones** | Devolución → escanear/EAN → cerrar con Efectivo |
| **Descuento (importe)** | Artículo/PLU o Subtotal → importe descuento → tecla Descuento importe |
| **Descuento (%)** | Artículo/PLU o Subtotal → % → tecla Descuento |
| **Cliente en espera** | „Cliente en espera“ / „Continuar cliente“ |
| **Ventas a crédito** | N.º cliente → Cliente → escanear → Subtotal → Crédito |
| **Pagos cliente (AKONTO)** | N.º cliente → Akonto → importe → Efectivo → ESC |
| **Cuenta cliente + / –** | Cambiar saldo sin efectivo: N.º cliente → Cuenta – → importe → Cuenta – |
