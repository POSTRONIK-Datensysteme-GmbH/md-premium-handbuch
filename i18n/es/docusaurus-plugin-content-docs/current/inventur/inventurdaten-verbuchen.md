# Contabilizar datos de inventario

Cómo editar y contabilizar los datos de inventario capturados después de [Inventory capture at till](/docs/inventur/inventurerfassung-kasse) o [Inventory capture with handheld](/docs/inventur/inventurerfassung-mde).

**Video:** [Post inventory data](https://youtu.be/_7ozr0G4xJY)

## Steps

### 1. Edit inventory articles

- Vaya a **Data maintenance** -> **Inventory** -> **Edit inventory articles**.
- Haga **doble clic** en el encabezado de fila para corregir la **cantidad real**.

### 2. Post differences

- Vaya a **Data maintenance** -> **Inventory** -> **Post differences**.
- Aquí se cierra el inventario para los artículos capturados y se contabiliza el stock correcto.

**Nota:** Ejecute "Post differences" solo después de que los artículos estén totalmente inventariados (almacén principal y secundario).
