# Crear lista de inventario por fecha de corte

Cómo crear una lista de inventario basada en una fecha de corte.

## Steps

1. **Master Data** → **Articles** → **Selection** (flecha abajo) → **Articles for inventory print**.
   En **Articles** elija Selection (flecha abajo) → **Articles for inventory print**. (Se seleccionan los artículos – espere.)
2. **Extras** → **Inventory** → **Inventory list by cut-off date**.
   En el menú de ventana elija **Inventory** → **Inventory list by cut-off date**.
3. Para **Inventory by cut-off date** se abre una ventana pequeña: haga clic en **Recalculate data** y confirme con **OK**. En la siguiente ventana introduzca la fecha para el cálculo del inventario (p. ej. 31.12.20…) y confirme con **OK**.
