# Restablecer datos de inventario del almacén

Cómo restablecer los datos de inventario de un almacén.

**Importante:** Esto **elimina el inventario actual** y restablece los datos. Hágalo **solo** cuando el inventario **no deba** contabilizarse.

## Procedure

- **Data maintenance** → **Inventory** → **Reset inventory data for main warehouse**
