# İade çağrısını el cihazına gönderme

İade çağrısını el cihazına nasıl göndereceğiniz.

## Adımlar

1. **Gazeteler** -> **Yeni iade çağrıları**'na gidin.
2. İade satırına **sağ tıklayın** ve bağlam menüsünden **MDE bayrağını ayarla**'yı seçin.
3. El cihazına aktarım başarılı olduğunda **Tarih** alanına bir **zaman damgası** yazılır.
