# Gazete rezervasyon listesi

Tüm rezervasyonları tek listede nasıl görüntüleyeceğiniz.

**Video:** [Newspaper reservation list](https://youtu.be/zhQWNf9jfNw)

## Adımlar

1. **Müşteri siparişleri** -> **Müşterilere sipariş**'e gidin.
2. Form araç çubuğunda **Seçim** simgesine tıklayın.
3. **Dizin listesi** simgesine tıklayın veya **F2** tuşuna basın.

Tüm rezervasyonlar görüntülenir.
