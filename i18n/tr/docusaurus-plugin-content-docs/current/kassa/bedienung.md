# XMPS Kullanım

XMPS kasa yazılımı kullanım konularının özeti. Ayrıntılı kılavuz ve videolar: Tüm adımlar bu kılavuzda açıklanmıştır.

| Konu | Kısa açıklama |
|------|---------------|
| **Giriş / Çıkış** | Kasiyer olarak kasa kullanımı |
| **Satışta miktar** | Miktar → „X“ tuşu → tarama veya PLU |
| **Manuel giriş (PLU)** | PLU veya EAN gir → PLU/EAN tuşu |
| **İade** | İade tuşu → ürünü tekrar gir (sadece son ürün) |
| **Fiş iptali** | İşlemi iptal et; tüm ürünler iade |
| **Para girişi / çıkışı** | „FUNKTION“ → tutar → Giriş veya Çıkış |
| **Çekmece aç / Satış yok** | Kayıt olmadan çekmece aç |
| **Kasa ağı onarımı** | „X“ modu → 4301 → Ara toplam → rapor → REG 1 |
| **Ürün iadeleri** | İade → tarama/EAN → Nakit ile kapat |
| **İndirim (tutar)** | Ürün/PLU veya Ara toplam → indirim tutarı → İndirim tutarı |
| **İndirim (%)** | Ürün/PLU veya Ara toplam → % → İndirim |
| **Müşteri bekle** | „Müşteri bekle“ / „Müşteriye devam“ |
| **Kredili satış** | Müşteri no → Müşteri → tarama → Ara toplam → Kredi |
| **Müşteri ödemeleri (AKONTO)** | Müşteri no → Akonto → tutar → Nakit → ESC |
| **Müşteri hesabı + / –** | Müşteri no → Hesap – → tutar → Hesap – |
