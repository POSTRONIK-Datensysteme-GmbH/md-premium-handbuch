# XMPS Kullanım

XMPS kasa yazılımı kullanım konularının özeti. Ayrıntılı kılavuz ve videolar: **POStronik Destek** → [Kullanım](https://support.postronik.at/help/de-de/14-bedienung).

| Konu | Kısa açıklama | Kılavuz |
|------|---------------|---------|
| **Giriş / Çıkış** | Kasiyer olarak kasa kullanımı | [Kullanım (genel)](https://support.postronik.at/help/de-de/14-bedienung) |
| **Satışta miktar** | Aynı ürünün çoklu satışı: miktar gir → „X“ tuşu → tarama veya PLU | [Miktar](https://support.postronik.at/help/de-de/14-bedienung/54-multiplikation-bei-verkaufsvorgangen) · [Video](https://youtu.be/JbhuffXc7ss) |
| **Manuel giriş (PLU)** | PLU veya EAN gir → PLU/EAN tuşu | [PLU tuşu](https://support.postronik.at/help/de-de/14-bedienung/52-manuelle-verbuchung-von-artikeln-uber-plu-taste) · [Video](https://youtu.be/nz-v3yCnvZA) |
| **İade** | İade tuşu → iade edilecek ürünü tekrar gir (sadece son ürün) | [İade](https://support.postronik.at/help/de-de/14-bedienung/57-storno) · [Video](https://youtu.be/WCb3dXabpJM) |
| **Fiş iptali** | İşlemi iptal et; tüm ürünler iade; örn. birkaç hata sonrası | [Fiş iptali](https://support.postronik.at/help/de-de/14-bedienung/58-bon-abbruch) · [Video](https://youtu.be/UGTz2CSr82E) |
| **Para girişi / çıkışı** | „FUNKTION“ ekranı → tutar → Giriş veya Çıkış (örn. kapıda ödeme) | [Giriş/Çıkış](https://support.postronik.at/help/de-de/14-bedienung/55-aus-oder-einzahlung-buchen) · [Video](https://youtu.be/syl1nxaN7A0) |
| **Çekmece aç / Satış yok** | Kayıt olmadan çekmece aç (örn. para üstü) | [Çekmece aç](https://support.postronik.at/help/de-de/14-bedienung/56-lade-offnen-kein-verkauf) |
| **Kasa ağı onarımı** | Ağ kesintisi sonrası: „X“ modu → 4301 → Ara toplam → rapor yazdır → REG 1 | [Kasa ağı](https://support.postronik.at/help/de-de/14-bedienung/24-kassenverbund-wiederherstellen) |
| **Ürün iadeleri** | İade → tarama/EAN → Nakit ile kapat (stok ve toplam düzeltmesi) | [İadeler](https://support.postronik.at/help/de-de/14-bedienung/59-artikel-retouren) |
| **İndirim (tutar)** | Ürün/PLU veya Ara toplam → indirim tutarı → İndirim tutarı tuşu | [İndirim tutarı](https://support.postronik.at/help/de-de/14-bedienung/63-nachlass-betrag) |
| **İndirim (%)** | Ürün/PLU veya Ara toplam → % → İndirim tuşu (önceden ayarlı olabilir) | [İndirim %](https://support.postronik.at/help/de-de/14-bedienung/64-rabatt) |
| **Müşteri bekle** | İşlemi duraklat, başka müşteriye geç → „Müşteri bekle“ / „Müşteriye devam“ | [Müşteri bekle](https://support.postronik.at/help/de-de/14-bedienung/65-kunde-parken) |
| **Kredili satış** | Müşteri no → Müşteri → tarama → Ara toplam → Kredi | [Kredili satış](https://support.postronik.at/help/de-de/14-bedienung/66-krediteinkaufe-mit-kundennummer) |
| **Müşteri ödemeleri (AKONTO)** | Müşteri no → Akonto → tutar → Nakit → ESC | [AKONTO](https://support.postronik.at/help/de-de/14-bedienung/67-kundeneinzahlungen-mit-akonto) |
| **Müşteri hesabı + / –** | Nakitsiz bakiye değişikliği (örn. havale): Müşteri no → Hesap – → tutar → Hesap – | [Müşteri hesabı +/–](https://support.postronik.at/help/de-de/14-bedienung/68-kundenkonto-kundenkonto) |
