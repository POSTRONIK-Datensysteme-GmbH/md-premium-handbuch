# XMPS Kapanışlar

XMPS kasa kapanış konularının özeti. Ayrıntılı kılavuz ve videolar: **POStronik Destek** → [Kapanışlar](https://support.postronik.at/help/de-de/15-abschlusse).

| Konu | Kısa açıklama | Kılavuz |
|------|---------------|---------|
| **Açılış bildirimi** | Vardiya başında açılış bakiyesini bildir; girişten sonra | [Açılış](https://support.postronik.at/help/de-de/15-abschlusse/47-wechselgelddeklaration) · [Video](https://youtu.be/7mZSyFPkba8) |
| **Para çekme** | Çekilen paraları bildir (banka yatırma); gün sonuna kadar toplanır | [Para çekme](https://support.postronik.at/help/de-de/15-abschlusse/48-geldabschopfung) · [Video](https://youtu.be/DibTxGMyNR8) |
| **Sayım** | Çekmece içeriğini bildir; istenen zaman, örn. vardiya sonu | [Sayım](https://support.postronik.at/help/de-de/15-abschlusse/49-geldzahlung) · [Video](https://youtu.be/F2qFLIu8zEQ) |
| **Sıfır fişi** | „FUNKT“ ekranı → X/Z → EVET (X1 modu) → 9811 [EVET]; örn. denetim için | [Sıfır fişi](https://support.postronik.at/help/de-de/15-abschlusse/1-nullbeleg) · [Video](https://youtu.be/G5M6xz0H0A4) |
| **Yıllık fiş** | „FUNKT“ ekranı → X/Z → EVET (X1 modu) → 9833 [EVET]; FinanzOnline’a bildir | [Yıllık fiş](https://support.postronik.at/help/de-de/15-abschlusse/3-jahresbeleg) · [Video](https://youtu.be/DDohE6W_aVE) |
| **Bankomat gün sonu** | Terminal için günlük gerekli (gerekirse TES kartı); Fonksiyon → X/Z → Z → EVET → 9898 → Ara toplam | [Bankomat gün sonu](https://support.postronik.at/help/de-de/15-abschlusse/69-bankomat-tagesabschluss) |
