# XMPS Kapanışlar

XMPS kasa kapanış konularının özeti. Tüm adımlar bu kılavuzda açıklanmıştır.

| Konu | Kısa açıklama |
|------|---------------|
| **Açılış bildirimi** | Vardiya başında; girişten sonra |
| **Para çekme** | Çekilen paraları bildir (banka yatırma); gün sonuna kadar toplanır |
| **Sayım** | Çekmece içeriğini bildir; istenen zaman, örn. vardiya sonu |
| **Sıfır fişi** | „FUNKT“ ekranı → X/Z → EVET (X1 modu) → 9811 [EVET]; örn. denetim için |
| **Yıllık fiş** | „FUNKT“ ekranı → X/Z → EVET (X1 modu) → 9833 [EVET]; FinanzOnline’a bildir |
| **Bankomat gün sonu** | Terminal için günlük gerekli (gerekirse TES kartı); Fonksiyon → X/Z → Z → EVET → 9898 → Ara toplam |
