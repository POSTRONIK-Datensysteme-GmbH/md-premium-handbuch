# Kasa (XMPS)

POStronik **XMPS kasa yazılımı** kılavuzları: satışta kullanım ve kapanış işlemleri.

Tüm adımlar bu kılavuzda açıklanmıştır.

## Kullanım

Günlük kasa işlemleri: giriş, ürün kaydı, iade, iadeler, indirimler, müşteri hesapları vb.

→ [Kullanım – tüm konular](/docs/kassa/bedienung)

## Kapanışlar

Kasa açılış bildirimi, para çekme, sayım, sıfır fişi, yıllık fiş ve bankomat gün sonu.

→ [Kapanışlar – tüm konular](/docs/kassa/abschluesse)
