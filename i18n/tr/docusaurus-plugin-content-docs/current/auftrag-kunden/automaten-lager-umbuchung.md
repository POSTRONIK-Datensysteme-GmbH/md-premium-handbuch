# Otomat depo transferi

Ürünleri otomat deposuna nasıl transfer edeceğiniz.

## Adımlar

1. **Müşteri siparişleri** -> **Müşterilere sipariş**'e gidin. Araç çubuğunda **Yeni kayıt**'a tıklayın. Yeni bir müşteri belgesi oluşturulur; imleç **Matchcode**'dadır.
2. **Matchcode** yanındaki **...**'e tıklayın. **ENTER** veya çift tıklama ile **otomatı** (müşteri) seçin.
3. **Hedef depo** yanındaki **...**'e tıklayın. **ENTER** veya çift tıklama ile **otomat deposunu** seçin.
4. Müşteri belgesini **Kaydet** simgesi veya **F10** ile kaydedin.

### Satır kalemleri ekleme

5. Satır listesi araç çubuğunda **Yeni kayıt**'a tıklayın. **RefNr** alanı aktiftir.
6. **RefNr**'de tedarikçi ürün numarası veya barkod girin (veya tarayın). Birden fazla eşleşmede doğru ürünü seçin.
7. **Miktar** girin.
8. Satırı **Kaydet** simgesi veya **F10** ile kaydedin. Diğer tüm satırları aynı şekilde girin.
9. Tüm satırlardan sonra: **Ekstralar** **[4]** altında **Pozisyonları serbest bırak** **[5]**'e tıklayın.
10. **Belge yazdır** **[1]** altında **Depo transferi** **[2]**'ye tıklayın. İletişim kutusunu **Tamam** ile onaylayın.
