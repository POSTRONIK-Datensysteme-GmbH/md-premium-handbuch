# Belirli tarihten beri envanteri olmayan ürünler – Stoğu temizleme

Belirli bir tarihten beri envanteri olmayan (artık stokta olmayan) ürünlerin stoğunu 0'a nasıl ayarlayacağınız.

**Video:** [Clear stock](https://youtu.be/oAPw1HhXhxs)

## Adımlar

1. **Ana veriler** → **Ürünler** → **Seçim** (ok aşağı) → **Belirli tarihten beri envanteri olmayan ürünler**.  
   Envanteri olmayan (artık stokta olmayan) tüm ürünler seçilir.
2. **Ana veriler** → **Ürünler** → **Ekstralar** → **Envanter** → **Envanter verisi ayarla**.  
   Önceki adımda seçilen tüm ürünler envanter durumuna ayarlanır.
3. **Veri bakımı** → **Envanter** → **Farkları kaydet**.  
   Seçilen ürünlerin stoğu **0**'a ayarlanır.
