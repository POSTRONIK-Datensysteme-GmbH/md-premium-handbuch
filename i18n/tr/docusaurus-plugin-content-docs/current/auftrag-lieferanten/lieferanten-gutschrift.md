# Tedarikçi iade fişi

Tedarikçi iade fişi nasıl oluşturulur.

**Video:** [Supplier credit note](https://youtu.be/tJLJ4UsYTuE)

## Adımlar

1. [Tedarikçi belgesi kaydetme](/docs/auftrag-lieferanten/lieferantenbeleg-erfassen) bölümünde anlatıldığı gibi tedarikçi belgesini kaydedin.
2. Pencere menüsünde **Belge yazdır** → **İade fişi**'ni seçin.
3. İade fişini tamamlamak için **Ekran**'a tıklayın.
