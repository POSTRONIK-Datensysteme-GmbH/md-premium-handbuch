# Tedarikçi siparişi

Tedarikçi siparişi oluşturma ve yazdırma.

**Video:** [Supplier order](https://youtu.be/T7lyD6STjRw)

## Adımlar

1. [Tedarikçi belgesi kaydetme](/docs/auftrag-lieferanten/lieferantenbeleg-erfassen) bölümünde anlatıldığı gibi tedarikçi belgesini kaydedin.
2. **[1] Belge yazdır**'a tıklayın ve **[2] Sipariş**'i seçin.
3. İletişim kutusunda sipariş tarihini girebilirsiniz.
4. **Ekran** ile onaylayın. Sipariş onayı penceresi açılır.
