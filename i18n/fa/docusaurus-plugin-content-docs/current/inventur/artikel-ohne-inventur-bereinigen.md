# مقالات بدون موجودی‌گیری از تاریخ – پاکسازی موجودی

نحوهٔ تنظیم موجودی روی ۰ برای مقالاتی که از یک تاریخ مشخص موجودی‌گیری نشده‌اند (دیگر در انبار نیستند).

**ویدیو:** [پاکسازی موجودی](https://youtu.be/oAPw1HhXhxs)

## مراحل

1. **داده‌های اصلی** → **مقالات** → **انتخاب** (فلش پایین) → **مقالات بدون موجودی‌گیری از تاریخ**.  
   این همهٔ مقالاتی را که موجودی‌گیری ندارند (دیگر در انبار نیستند) انتخاب می‌کند.
2. **داده‌های اصلی** → **مقالات** → **افزونه‌ها** → **موجودی‌گیری** → **تنظیم داده‌های موجودی**.  
   این همهٔ مقالات انتخاب‌شده در مرحلهٔ قبل را به وضعیت موجودی تنظیم می‌کند.
3. **نگهداری داده** → **موجودی‌گیری** → **ثبت تفاوت‌ها**.  
   موجودی مقالات انتخاب‌شده روی **۰** تنظیم می‌شود.
