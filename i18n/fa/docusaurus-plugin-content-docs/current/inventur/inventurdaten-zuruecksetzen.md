# بازنشانی داده‌های موجودی انبار

نحوهٔ بازنشانی داده‌های موجودی برای یک انبار.

**مهم:** این **موجودی فعلی را حذف** می‌کند و داده را بازنشانی می‌کند. این کار را **فقط** وقتی انجام دهید که موجودی **نباید** ثبت شود.

## روش

- **نگهداری داده** → **موجودی‌گیری** → **بازنشانی داده‌های موجودی انبار اصلی**
