# ثبت داده‌های موجودی

نحوهٔ ویرایش و ثبت داده‌های موجودی ثبت‌شده پس از [ثبت موجودی در صندوق](/docs/inventur/inventurerfassung-kasse) یا [ثبت موجودی با دستگاه دستی](/docs/inventur/inventurerfassung-mde).

**ویدیو:** [ثبت داده‌های موجودی](https://youtu.be/_7ozr0G4xJY)

## مراحل

### ۱. ویرایش مقالات موجودی

- به **نگهداری داده** -> **موجودی‌گیری** -> **ویرایش مقالات موجودی** بروید.
- روی هدر ردیف **دوبار کلیک** کنید تا **تعداد واقعی** را اصلاح کنید.

### ۲. ثبت تفاوت‌ها

- به **نگهداری داده** -> **موجودی‌گیری** -> **ثبت تفاوت‌ها** بروید.
- اینجا موجودی برای مقالات ثبت‌شده بسته می‌شود و موجودی صحیح ثبت می‌شود.

**نکته:** «ثبت تفاوت‌ها» را فقط بعد از موجودی‌گیری کامل مقالات (انبار اصلی و فرعی) اجرا کنید.
