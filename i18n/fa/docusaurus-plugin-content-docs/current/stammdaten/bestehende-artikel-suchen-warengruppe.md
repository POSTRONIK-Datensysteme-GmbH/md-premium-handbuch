# جستجوی مقالات موجود با گروه کالا

نحوهٔ یافتن همهٔ مقالات در یک گروه کالای مشخص.

**ویدیو:** [جستجوی مقالات موجود با گروه کالا](https://youtu.be/jBWbNb3AjcE)

## مراحل

1. فرم مقاله را باز کنید: **داده‌های اصلی** → **مقالات**.
2. روی **آیکون پنجرهٔ انتخاب** کنار **WGR** کلیک کنید و دکمهٔ **همه** را در پنجرهٔ جدید انتخاب کنید.
3. گروه کالا را با **دوبار کلیک** روی هدر ردیف آن انتخاب کنید.
4. **آیکون جستجو** یا **F3** را برای شروع جستجو بزنید.
5. سیستم همهٔ مقالات آن گروه را فهرست می‌کند. از کلیدهای جهت‌نما یا **نمایش لیست** (**F2**) برای مرور استفاده کنید.
6. روی هدر ستون کلیک کنید تا مرتب شود.
7. روی هدر ردیف **دوبار کلیک** کنید تا نمای جزئیات مقاله باز شود.
8. لیست را با **X** یا **ESC** ببندید.
