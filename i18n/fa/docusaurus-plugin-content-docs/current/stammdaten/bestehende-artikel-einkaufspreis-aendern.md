# تغییر قیمت خرید مقالهٔ موجود

نحوهٔ تغییر قیمت خرید یک مقالهٔ موجود.

**ویدیو:** [تغییر قیمت خرید](https://youtu.be/sNBcAntVThI)

## مراحل

1. مقاله را طبق [جستجوی مقالات موجود با بارکد](/docs/stammdaten/bestehende-artikel-suchen-barcode) (یا با شرح/گروه کالا) پیدا کنید.
2. تب **(۱) خرید** را بزنید. قیمت خرید فعلی نمایش داده می‌شود.
3. قیمت قدیمی را در **(۲) مبلغ** (قیمت خرید خالص) انتخاب و بازنویسی کنید.
4. آیکون **(۳) ذخیره** را بزنید یا **F10** را برای ذخیره فشار دهید.
