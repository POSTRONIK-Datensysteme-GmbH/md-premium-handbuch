# تغییر قیمت فروش مقالهٔ موجود

نحوهٔ تغییر قیمت فروش یک مقالهٔ موجود.

**ویدیو:** [تغییر قیمت فروش](https://youtu.be/yzCTto9EC4I)

## مراحل

1. مقاله را طبق [جستجوی مقالات موجود با بارکد](/docs/stammdaten/bestehende-artikel-suchen-barcode) (یا با شرح/گروه کالا) پیدا کنید.
2. تب **(۱) قیمت‌ها** را بزنید. قیمت فعلی نمایش داده می‌شود.
3. قیمت قدیمی را در **(۲) مبلغ شامل** انتخاب و بازنویسی کنید.
4. آیکون **(۳) ذخیره** را بزنید یا **F10** را برای ذخیره فشار دهید.
