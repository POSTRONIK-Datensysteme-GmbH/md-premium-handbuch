# کار با صندوق XMPS

خلاصه موضوعات کار با صندوق XMPS. همه مراحل در این راهنما توضیح داده شده‌اند.

| موضوع | توضیح کوتاه |
|--------|--------------|
| **ورود / خروج** | استفاده از صندوق به‌عنوان صندوقدار |
| **تعداد در فروش** | چند قلم یکسان: وارد کردن تعداد → دکمه „X“ → اسکن یا PLU |
| **ثبت دستی (PLU)** | وارد کردن PLU یا EAN → دکمه PLU/EAN |
| **استorno** | دکمه Storno → ثبت دوباره قلم برای استorno (فقط آخرین قلم) |
| **لغو رسید** | لغو تراکنش؛ همه اقلام استorno |
| **واریز / برداشت** | صفحه „FUNKTION“ → مبلغ → واریز یا برداشت |
| **باز کردن کشو / بدون فروش** | باز کردن کشو بدون ثبت |
| **بازگردانی شبکه صندوق‌ها** | حالت „X“ → 4301 → جمع جزء → گزارش → REG 1 |
| **مرجوعی** | مرجوعی → اسکن/EAN → بستن با نقد |
| **تخفیف (مبلغ)** | کالا/PLU یا جمع جزء → مبلغ تخفیف → دکمه تخفیف مبلغ |
| **تخفیف (%)** | کالا/PLU یا جمع جزء → درصد → دکمه تخفیف |
| **مشتری در انتظار** | „مشتری در انتظار“ / „ادامه مشتری“ |
| **فروش نسیه** | شماره مشتری → مشتری → اسکن → جمع جزء → نسیه |
| **پرداخت مشتری (AKONTO)** | شماره مشتری → آکونتو → مبلغ → نقد → ESC |
| **حساب مشتری + / –** | شماره مشتری → حساب – → مبلغ → حساب – |
