# کار با صندوق XMPS

خلاصه موضوعات کار با صندوق XMPS. راهنما و ویدیوهای تفصیلی: **پشتیبانی POStronik** → [کار با صندوق](https://support.postronik.at/help/de-de/14-bedienung).

| موضوع | توضیح کوتاه | راهنما |
|--------|--------------|--------|
| **ورود / خروج** | استفاده از صندوق به‌عنوان صندوقدار | [کار با صندوق (خلاصه)](https://support.postronik.at/help/de-de/14-bedienung) |
| **تعداد در فروش** | چند قلم یکسان: وارد کردن تعداد → دکمه „X“ → اسکن یا PLU | [تعداد](https://support.postronik.at/help/de-de/14-bedienung/54-multiplikation-bei-verkaufsvorgangen) · [ویدیو](https://youtu.be/JbhuffXc7ss) |
| **ثبت دستی (PLU)** | وارد کردن PLU یا EAN → دکمه PLU/EAN | [دکمه PLU](https://support.postronik.at/help/de-de/14-bedienung/52-manuelle-verbuchung-von-artikeln-uber-plu-taste) · [ویدیو](https://youtu.be/nz-v3yCnvZA) |
| **استorno** | دکمه Storno → ثبت دوباره قلم برای استorno (فقط آخرین قلم) | [استorno](https://support.postronik.at/help/de-de/14-bedienung/57-storno) · [ویدیو](https://youtu.be/WCb3dXabpJM) |
| **لغو رسید** | لغو تراکنش؛ همه اقلام استorno؛ مثلاً بعد از چند خطا | [لغو رسید](https://support.postronik.at/help/de-de/14-bedienung/58-bon-abbruch) · [ویدیو](https://youtu.be/UGTz2CSr82E) |
| **واریز / برداشت** | صفحه „FUNKTION“ → مبلغ → واریز یا برداشت (مثلاً پرداخت در محل) | [واریز/برداشت](https://support.postronik.at/help/de-de/14-bedienung/55-aus-oder-einzahlung-buchen) · [ویدیو](https://youtu.be/syl1nxaN7A0) |
| **باز کردن کشو / بدون فروش** | باز کردن کشو بدون ثبت (مثلاً خرد) | [باز کردن کشو](https://support.postronik.at/help/de-de/14-bedienung/56-lade-offnen-kein-verkauf) |
| **بازگردانی شبکه صندوق‌ها** | بعد از قطع شبکه: حالت „X“ → 4301 → جمع جزء → چاپ گزارش → REG 1 | [شبکه صندوق‌ها](https://support.postronik.at/help/de-de/14-bedienung/24-kassenverbund-wiederherstellen) |
| **مرجوعی** | مرجوعی → اسکن/EAN → بستن با نقد (اصلاح موجودی و جمع) | [مرجوعی](https://support.postronik.at/help/de-de/14-bedienung/59-artikel-retouren) |
| **تخفیف (مبلغ)** | کالا/PLU یا جمع جزء → مبلغ تخفیف → دکمه تخفیف مبلغ | [تخفیف مبلغ](https://support.postronik.at/help/de-de/14-bedienung/63-nachlass-betrag) |
| **تخفیف (%)** | کالا/PLU یا جمع جزء → درصد → دکمه تخفیف (ممکن است از قبل تنظیم باشد) | [تخفیف %](https://support.postronik.at/help/de-de/14-bedienung/64-rabatt) |
| **مشتری در انتظار** | توقف تراکنش، خدمت به مشتری دیگر → „مشتری در انتظار“ / „ادامه مشتری“ | [مشتری در انتظار](https://support.postronik.at/help/de-de/14-bedienung/65-kunde-parken) |
| **فروش نسیه** | شماره مشتری → مشتری → اسکن → جمع جزء → نسیه | [فروش نسیه](https://support.postronik.at/help/de-de/14-bedienung/66-krediteinkaufe-mit-kundennummer) |
| **پرداخت مشتری (AKONTO)** | شماره مشتری → آکونتو → مبلغ → نقد → ESC | [AKONTO](https://support.postronik.at/help/de-de/14-bedienung/67-kundeneinzahlungen-mit-akonto) |
| **حساب مشتری + / –** | تغییر مانده بدون نقد (مثلاً حواله): شماره مشتری → حساب – → مبلغ → حساب – | [حساب مشتری +/–](https://support.postronik.at/help/de-de/14-bedienung/68-kundenkonto-kundenkonto) |
