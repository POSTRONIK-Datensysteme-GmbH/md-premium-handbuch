# بستن روز XMPS

خلاصه موضوعات بستن روز صندوق XMPS. همه مراحل در این راهنما توضیح داده شده‌اند.

| موضوع | توضیح کوتاه |
|--------|--------------|
| **اعلام موجودی اولیه** | در شروع شیفت؛ بعد از ورود |
| **برداشت نقد** | اعلام برداشت‌ها؛ تا پایان روز جمع می‌شود |
| **شمارش** | اعلام محتوای فعلی کشو؛ مثلاً پایان شیفت |
| **رسید صفر** | „FUNKT“ → X/Z → بله (X1) → 9811 [بله] |
| **رسید سالانه** | „FUNKT“ → X/Z → بله (X1) → 9833 [بله]؛ FinanzOnline |
| **بستن روز پایانه بانکی** | تابع → X/Z → Z → بله → 9898 → جمع جزء |
