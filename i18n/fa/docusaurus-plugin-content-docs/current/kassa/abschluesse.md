# بستن روز XMPS

خلاصه موضوعات بستن روز صندوق XMPS. راهنما و ویدیو: **پشتیبانی POStronik** → [بستن روز](https://support.postronik.at/help/de-de/15-abschlusse).

| موضوع | توضیح کوتاه | راهنما |
|--------|--------------|--------|
| **اعلام موجودی اولیه** | اعلام موجودی اولیه در شروع شیفت؛ بعد از ورود | [موجودی اولیه](https://support.postronik.at/help/de-de/15-abschlusse/47-wechselgelddeklaration) · [ویدیو](https://youtu.be/7mZSyFPkba8) |
| **برداشت نقد** | اعلام برداشت‌ها (واریز به بانک)؛ تا پایان روز جمع می‌شود | [برداشت](https://support.postronik.at/help/de-de/15-abschlusse/48-geldabschopfung) · [ویدیو](https://youtu.be/DibTxGMyNR8) |
| **شمارش** | اعلام محتوای فعلی کشو؛ هر زمان، مثلاً پایان شیفت | [شمارش](https://support.postronik.at/help/de-de/15-abschlusse/49-geldzahlung) · [ویدیو](https://youtu.be/F2qFLIu8zEQ) |
| **رسید صفر** | صفحه „FUNKT“ → X/Z → بله (حالت X1) → 9811 [بله]؛ مثلاً برای ممیزی | [رسید صفر](https://support.postronik.at/help/de-de/15-abschlusse/1-nullbeleg) · [ویدیو](https://youtu.be/G5M6xz0H0A4) |
| **رسید سالانه** | صفحه „FUNKT“ → X/Z → بله (حالت X1) → 9833 [بله]؛ اعلام در FinanzOnline | [رسید سالانه](https://support.postronik.at/help/de-de/15-abschlusse/3-jahresbeleg) · [ویدیو](https://youtu.be/DDohE6W_aVE) |
| **بستن روز پایانه بانکی** | روزانه برای پایانه لازم (در صورت نیاز کارت TES)؛ تابع → X/Z → Z → بله → 9898 → جمع جزء | [بستن روز پایانه](https://support.postronik.at/help/de-de/15-abschlusse/69-bankomat-tagesabschluss) |
