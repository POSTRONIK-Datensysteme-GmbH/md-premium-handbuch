# Print labels

How to print product labels for articles.

**Video:** [Print labels](https://youtu.be/dhK3zqw0120)

## Steps

### 1. Find article and request labels

1. Find the article as in [Search existing articles by barcode](/docs/stammdaten/bestehende-artikel-suchen-barcode) (or by description/product group).
2. Open the **(1) Extras** tab and choose **(2) Request product labels**.
3. Enter the number of labels in the new window and confirm with **(3) OK**. Repeat if you need more.
4. When all labels are requested, click **Print product labels**.

### 2. Print labels

1. Select the **(1) Label format** and click **(2) Labels for all articles**.
2. Enter the number of **(3) Blank labels** if needed and click **(4) Print**.
3. In the print preview, click the **(1) Print** icon top-left, choose the printer and confirm with **OK**. After printing, confirm whether the label file can be deleted and close the label window.
