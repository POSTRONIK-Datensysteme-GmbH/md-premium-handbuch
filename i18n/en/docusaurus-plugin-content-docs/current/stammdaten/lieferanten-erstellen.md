# Create supplier

How to create a supplier.

## Steps

1. In the main menu, select **Master Data** → **Suppliers**. The **Suppliers** window opens and shows the last opened supplier. For detailed descriptions see also MDLIE.

   ![Supplier form](/img/docs/stammdaten-lieferanten-erstellen-1.png)

2. Click **New record** (![New record](/img/docs/fw_newbutton.png)) to create a new supplier. The **Lie Id** field in the **ID** group is automatically filled with the next available supplier number:

   ![Lie Id auto-filled](/img/docs/stammdaten-lieferanten-erstellen-2.png)

3. In the **Match** (MATCHCODE) field, enter a search term that will help you find the supplier quickly later, e.g. the supplier’s company name.

4. In the **Adressart** (address type) field, choose whether the supplier is a company, a female or male person, or whether address type/gender is unknown.

   :::info
   The address type affects, among other things, the salutation in outgoing correspondence.
   :::

5. Fill in all other fields in the **ID** group that are relevant for you.

   ![ID group](/img/docs/stammdaten-lieferanten-erstellen-3.png)

6. Enter the supplier’s address data in the **Adresse** (Address) group.

   ![Address group](/img/docs/stammdaten-lieferanten-erstellen-4.png)

7. Enter the supplier’s contact data in the **Kontakt** (Contact) group.

   ![Contact group](/img/docs/stammdaten-lieferanten-erstellen-5.png)

8. When you have entered all information relevant to you for the supplier, click **Save** (![Save](/img/docs/fw_savebutton.png)).
