# Search existing articles by description

How to find an article in the database by its description.

**Video:** [Search existing articles by description](https://youtu.be/mbJhea9m2yk)

## Steps

1. Open the article form: **Master Data** → **Articles**.
2. Press the **search icon** or key **F3** to start the search.
3. In the **Matchcode** field, enter the article description with **%** at the start and end (e.g. `%MARLB%`).
4. The system lists all articles containing that text. Use the arrow keys to browse or click **List view** (or **F2**) to see them in a table.
5. Click a column header to sort by that column.
6. **Double-click** a row header to open the article detail view.
7. Close the list view by clicking **X** or pressing **ESC**.
