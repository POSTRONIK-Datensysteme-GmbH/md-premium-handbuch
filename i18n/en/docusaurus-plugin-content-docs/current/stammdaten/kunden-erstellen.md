# Create customer

How to create a customer manually.

## Steps

1. In the main menu, select **Master Data** → **Customers**. The **Customers** window opens and shows the last opened customer. For detailed descriptions see also MDKND.

   ![Customer form](/img/docs/stammdaten-kunden-erstellen-1.png)

2. Click **New record** (![New record](/img/docs/fw_newbutton.png)). The **Knd Id** field in the **ID** group is automatically filled with the next available customer number:

   ![Knd Id auto-filled](/img/docs/stammdaten-kunden-erstellen-2.png)

3. In the **Match** (MATCHCODE) field, enter a search term that will help you find the customer quickly later, e.g. the customer’s last name or company name.

4. In the **Adressart** (address type) field, choose whether the customer is a company, a female or male person, or whether address type/gender is unknown.

   :::info
   The address type affects, among other things, the salutation in outgoing correspondence.
   :::

5. Fill in all other fields in the **ID** group that are relevant for you.

   ![ID group](/img/docs/stammdaten-kunden-erstellen-3.png)

6. Enter the customer’s address data in the **Adresse** (Address) group.

   ![Address group](/img/docs/stammdaten-kunden-erstellen-4.png)

7. Enter the customer’s contact data in the **Kontakt** (Contact) group.

   ![Contact group](/img/docs/stammdaten-kunden-erstellen-5.png)

8. In the **#** group, edit the following fields and options as needed:
   - **Kennung:** Field for one or more predefined characters that change program behaviour. For general information see Kennungen. For a description of the characters that can be used see Kennungen: Kunden.

9. When you have entered all information relevant to you for the customer, click **Save** (![Save](/img/docs/fw_savebutton.png)).
