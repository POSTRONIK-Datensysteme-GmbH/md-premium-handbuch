# Create MD order

How to create a customer order and add line items (e.g. for newspaper reservation).

## Steps

1. In the menu go to **Customer orders** -> **Order customers**. The last edited customer document is shown. Click **New record** in the toolbar.
2. Click **...** next to **Matchcode** and select the customer from the list.
3. Save the customer document with the **save icon** or **F10**.

### Add line items

4. Click **New record** in the **line list** toolbar.
5. In **RefNr** enter a supplier article number or barcode (or scan). Select the article from the list.
6. Enter the **Quantity** in the quantity field.
7. Save the line with the **save icon** or **F10**.
8. Repeat steps 4–7 for more lines.

### Newspaper reservation

9. Under **Document print** click **Newspaper reservation order** and confirm the window with **OK**.
