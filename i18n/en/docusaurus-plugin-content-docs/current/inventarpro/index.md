# InventarPro (MDE)

**InventarPro** is the app for the **MDE device** (mobile data capture) in MD-Premium merchandise management. With InventarPro you can capture delivery notes, inventory and other documents on the device and transfer data to your system.

## Topics

- [Quick start guide](/docs/inventarpro/schnellstart) – First steps, daily routine, main menu, sample delivery note, updates
- [User manual (full)](/docs/inventarpro/anleitung) – All modules, settings, import/export, new features

## Note

Ensure the MDE device is connected to the company Wi‑Fi. If you have issues, check the Wi‑Fi connection first and contact support if needed.
