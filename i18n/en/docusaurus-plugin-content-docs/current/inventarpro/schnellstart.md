# InventarPro – Quick start guide

This guide helps you with the first steps using **InventarPro** on the MDE device.

---

## 1. Preparation (one-time on first setup)

- **Connect Wi‑Fi:** Ensure the MDE device is connected to your company Wi‑Fi before starting the app.
- **First start:** Open the InventarPro app. It should log in automatically and show the main menu.
- **Load database for the first time:**
  - In the main menu tap the **Import** button.
  - Wait until the import is complete. **This step is essential!**

---

## 2. Daily routine before starting work

- **Update database:** Each day before first use, tap **Import** in the main menu so you have the latest article data, prices and supplier information.

---

## 3. Understanding the main menu

- **Tiles:** Each tile starts a task (e.g. delivery note, inventory).
- **Bottom buttons:**
  - **Import:** Loads data (see steps 1 & 2).
  - **Article info (i):** Shows details for a scanned article.
  - **Export:** Sends completed documents to your system (blinks when data is ready).

---

## 4. Example: Recording a delivery note (goods receipt)

1. **Start module:** In the main menu tap the **Delivery note** tile.
2. **Enter header data:**
   - Select the supplier.
   - You can adjust delivery note number and date if needed (optional).
   - Tap **New**.
3. **Scan articles:**
   - Scan the barcode of the first article.
   - Scan each further article or adjust the quantity in the quantity field.
4. **Save document:** When all articles are scanned, press the device **Back** button. The delivery note appears in the list.
5. **Close document:** Select the delivery note in the list and tap **Close**.
6. **Export data:** Return to the main menu. When the **Export** button blinks, tap it to send the data.

---

## 5. App updates

- The app checks for updates on start. If a **Download** button appears, tap it and follow the installation instructions.

---

## If you have problems

Check the **Wi‑Fi connection** first. Otherwise contact support.
