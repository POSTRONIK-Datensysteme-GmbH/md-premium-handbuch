# Tobaccoland – Send order online

How to send an order to Tobaccoland.

## Steps

1. In the **Supplier orders** menu select a Tobaccoland order.
2. Under **Extras** click **Tobaccoland Order 2019**. The external program **PTTobOrder** for sending to Tobaccoland opens.
3. Click **SEND** to submit the order to Tobaccoland.

You will receive a confirmation e-mail from Tobaccoland.
