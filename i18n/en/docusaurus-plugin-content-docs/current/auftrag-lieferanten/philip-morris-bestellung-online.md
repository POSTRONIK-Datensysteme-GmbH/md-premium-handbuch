# Philip Morris – Send order online

How to send an order to Philip Morris.

## Steps

1. In the **Supplier orders** menu select a Philip Morris order.
2. Under **Extras** click **Philip Morris Order**. The external program **PTPmiOrder** for sending to Philip Morris opens.
3. Click **SEND** to submit the order to Philip Morris.

You will receive a confirmation e-mail from Philip Morris.
