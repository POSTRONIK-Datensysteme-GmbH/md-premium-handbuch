# Moosmayr – Send order online

How to send an order to Moosmayr.

## Steps

1. In the **Supplier orders** menu select a Moosmayr order.
2. Under **Extras** choose **Moosmayr Order Online**.
3. A pop-up appears with the order number.

**Variant 1:** The order was sent successfully.

**Variant 2:** If the message after "|live|" shows a **negative value** with an order number, that order could not be sent. The displayed order number cannot be ordered from Moosmayr – **this line must be removed from the order** before you can send it.
