# Supplier order from order suggestion

How to run an order suggestion and place the order.

**Video:** [Supplier order from order suggestion](https://youtu.be/tq-oQN8HqFE)

## Steps

1. In the menu bar click **Execute order suggestion**.
2. In the new window select the supplier and confirm with **OK**.

The suggestion is calculated. The footer shows number of items and purchase cost total.

### Adjust quantities and place order

3. To change suggested quantities: click in the **Quantity** field and enter the desired amount.
4. To place the order: in the window menu open **Extras** and choose **Order creation**. Confirm with **Yes**, wait for the process, then confirm the completion message. The program shows the order.

## Columns in the order suggestion

- **WGR:** Article product group  
- **Art_ID:** Internal article number (also order number)  
- **Match:** Article description  
- **Menge:** Order quantity  
- **Meh:** Unit (often case)  
- **LagerMe:** Current stock in order unit  
- **LagMEH:** Current stock in base unit (pack)  
- **MinBest:** Minimum stock  
- **Ekp:** Purchase price for order unit  
- **EKP-Summe / Vkp.Inkl-Sum:** Totals  
- **MengeVK7TA:** Packs sold in last 7 days  

## Special suppliers

- **Tobaccoland:** Window menu **Extras** → **Tobaccoland Order 2019** (see [Tobaccoland – Send order online](/docs/auftrag-lieferanten/tobaccoland-bestellung-online)).
- **Philip Morris:** **Extras** → **Philip Morris Order** (see [Philip Morris – Send order online](/docs/auftrag-lieferanten/philip-morris-bestellung-online)).
- **Moosmayr:** **Extras** → **Moosmayr Order Online** (see [Moosmayr – Send order online](/docs/auftrag-lieferanten/moosmayr-bestellung-online)).
- **Other tobacco suppliers (e.g. Mossmayr, Danzcek):** Send order suggestion by e-mail.
