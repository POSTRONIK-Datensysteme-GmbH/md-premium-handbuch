# Supplier order

How to create a supplier order and print it.

**Video:** [Supplier order](https://youtu.be/T7lyD6STjRw)

## Steps

1. Record the supplier document as in [Record supplier document](/docs/auftrag-lieferanten/lieferantenbeleg-erfassen).
2. Click **[1] Document print** and choose **[2] Order**.
3. In the dialog you can enter the order date.
4. Confirm with **Screen**. A window with the order confirmation opens.
