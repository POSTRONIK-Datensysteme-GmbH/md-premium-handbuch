# Pre-return from delivery notes

How to do pre-returns directly from delivered newspaper items.

## Steps

1. From the quick-access bar choose **Newspapers** -> **Pre-return from delivery notes**.
2. Select a **delivery date** range and click **Display**. Only deliveries in that period are shown.
3. The window shows all newspaper return deliveries in the period, including delivery, sales and return info for the last 10 issues per item.
4. Select a **delivery item** in the upper area. The lower area shows the issue quantity overview.

**Add item to return call** (an open return call must exist for the supplier):

- **Right-click** the item row -> **Add to return order...**, or
- **Double-click** the item row header.
5. In the dialog enter the **pre-return quantity** (suggestion: delivery quantity) and confirm with **OK**.

**Notes:** On add, the "commission date" is set to today (prevents change on "Set quantities to stock"). An item cannot be added if it is already in the return call with the same issue number.
