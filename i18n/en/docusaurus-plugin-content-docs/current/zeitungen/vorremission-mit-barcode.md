# Pre-return with barcode

How to add items by barcode/EAN to a return call.

## Prerequisite

An **open return call** must exist for the supplier.

## Steps

1. Click **Newspapers** → **Pre-return with barcode**.
2. Enter the EAN under **[1] RefNr** or scan with the barcode scanner. Confirm with **[2] Display**.
3. To add the item to a return call:
   - **Right-click** the item row → **Add to return order…**, or
   - **Double-click** the item row header.

The entered quantity is appended to the **most recent** return order of that supplier.

**Notes:** On add, the "commission date" is set to today. An item cannot be added if it is already in the return call with the same issue number.
