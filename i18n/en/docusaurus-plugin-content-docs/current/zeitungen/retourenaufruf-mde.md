# Send return call to handheld

How to send a return call to the handheld device.

## Steps

1. Go to **Newspapers** -> **New return calls**.
2. **Right-click** the return row and choose **Set MDE flag** in the context menu.
3. When transfer to the handheld succeeds, a **timestamp** is written in the **Date** field.
