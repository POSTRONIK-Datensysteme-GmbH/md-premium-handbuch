# Returns

After returns calls are imported from EH-2000 data, they are created as document type **AM** (return call). Return calls can be recalculated (quantities set to stock) and printed as often as needed. Newspaper articles can be added to existing return calls for pre-return – manually or from delivered items (see [Pre-return from delivery notes](/docs/zeitungen/vorremission-aus-lieferscheinen), [Pre-return with barcode](/docs/zeitungen/vorremission-mit-barcode)). After all return quantities are recorded, return calls are converted via document print into a **return** (order code **ZM**). Return documents are sent to the wholesaler automatically.

## Open and print return call

1. Go to **Newspapers** -> **New return calls**. The window shows all newspaper return calls with net totals.
2. Open a return call by **double-clicking** the row header. Ensure the **Articles** tab is selected.
3. Print via window menu **Document print** -> **Return order (AM)**.
4. Use the printout to find newspaper items in the sales and storage area. Enter the **actual count found** on the document. The document can be closed and reopened anytime.
