# Complete return call

When all quantities for a return call are recorded correctly, the return order must be converted into a **return**. The return is then sent to the wholesaler automatically.

## Steps

1. Start **Newspapers** -> **New return calls** and open a return order by **double-clicking** the row header.
2. Check the return order lines for correctness and completeness.
3. In the window menu choose **Document print** -> **Return**.
4. In the dialog choose **Screen** print and wait for the output.
5. The return is now complete.
