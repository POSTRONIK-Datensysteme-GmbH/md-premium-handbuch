# Post inventory data

How to edit and post captured inventory data after [Inventory capture at till](/docs/inventur/inventurerfassung-kasse) or [Inventory capture with handheld](/docs/inventur/inventurerfassung-mde).

**Video:** [Post inventory data](https://youtu.be/_7ozr0G4xJY)

## Steps

### 1. Edit inventory articles

- Go to **Data maintenance** -> **Inventory** -> **Edit inventory articles**.
- **Double-click** the row header to correct the **actual quantity**.

### 2. Post differences

- Go to **Data maintenance** -> **Inventory** -> **Post differences**.
- Here inventory is closed for the captured articles and the correct stock is posted.

**Note:** Run "Post differences" only after articles are fully inventoried (main and secondary warehouse).
