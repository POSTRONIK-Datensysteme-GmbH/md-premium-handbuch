# Create inventory list by cut-off date

How to create an inventory list based on a cut-off date.

## Steps

1. **Master Data** → **Articles** → **Selection** (arrow down) → **Articles for inventory print**.  
   Under **Articles** choose Selection (arrow down) → **Articles for inventory print**. (Articles are selected – please wait.)
2. **Extras** → **Inventory** → **Inventory list by cut-off date**.  
   In the window menu choose **Inventory** → **Inventory list by cut-off date**.
3. For **Inventory by cut-off date** a small window opens: click **Recalculate data** and confirm with **OK**. In the next window enter the date for the inventory calculation (e.g. 31.12.20…) and confirm with **OK**.
