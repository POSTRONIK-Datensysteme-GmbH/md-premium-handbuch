# Reset inventory data for warehouse

How to reset inventory data for a warehouse.

**Important:** This **deletes the current inventory** and resets the data. Do this **only** when the inventory **should not** be posted.

## Procedure

- **Data maintenance** → **Inventory** → **Reset inventory data for main warehouse**
