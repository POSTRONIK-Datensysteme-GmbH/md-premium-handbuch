# XMPS Operation

Overview of XMPS till software operation topics. Detailed guides and videos: **POStronik Support** → [Operation](https://support.postronik.at/help/de-de/14-bedienung).

| Topic | Short description | Guide |
|-------|-------------------|-------|
| **Log in / Log out** | Use till as cashier | [Operation (overview)](https://support.postronik.at/help/de-de/14-bedienung) |
| **Quantity in sales** | Multiple same items: enter quantity → press "X" → scan or enter PLU | [Quantity](https://support.postronik.at/help/de-de/14-bedienung/54-multiplikation-bei-verkaufsvorgangen) · [Video](https://youtu.be/JbhuffXc7ss) |
| **Manual entry (PLU)** | Enter PLU or EAN → PLU/EAN key | [PLU key](https://support.postronik.at/help/de-de/14-bedienung/52-manuelle-verbuchung-von-artikeln-uber-plu-taste) · [Video](https://youtu.be/nz-v3yCnvZA) |
| **Void** | Press Void → re-enter item to void (last item only) | [Void](https://support.postronik.at/help/de-de/14-bedienung/57-storno) · [Video](https://youtu.be/WCb3dXabpJM) |
| **Cancel receipt** | Cancel transaction, all items voided; e.g. after multiple errors | [Cancel receipt](https://support.postronik.at/help/de-de/14-bedienung/58-bon-abbruch) · [Video](https://youtu.be/UGTz2CSr82E) |
| **Pay in / Pay out** | "FUNKTION" screen → enter amount → Pay in or Pay out (e.g. COD) | [Pay in/out](https://support.postronik.at/help/de-de/14-bedienung/55-aus-oder-einzahlung-buchen) · [Video](https://youtu.be/syl1nxaN7A0) |
| **Open drawer / No sale** | Open cash drawer without registering (e.g. change) | [Open drawer](https://support.postronik.at/help/de-de/14-bedienung/56-lade-offnen-kein-verkauf) |
| **Restore till network** | After network issue: Mode "X" → 4301 → Subtotal → report prints → REG 1 | [Till network](https://support.postronik.at/help/de-de/14-bedienung/24-kassenverbund-wiederherstellen) |
| **Returns** | Return → scan/article EAN → complete with Cash (stock & totals correction) | [Returns](https://support.postronik.at/help/de-de/14-bedienung/59-artikel-retouren) |
| **Discount (amount)** | Article/PLU or Subtotal → enter discount amount → Discount amount key | [Discount amount](https://support.postronik.at/help/de-de/14-bedienung/63-nachlass-betrag) |
| **Discount (%)** | Article/PLU or Subtotal → enter % → Discount key (may be preset) | [Discount %](https://support.postronik.at/help/de-de/14-bedienung/64-rabatt) |
| **Park customer** | Pause transaction, serve another → "Park customer" / "Resume customer" | [Park customer](https://support.postronik.at/help/de-de/14-bedienung/65-kunde-parken) |
| **Credit sales** | Customer number → Customer → scan items → Subtotal → Credit | [Credit sales](https://support.postronik.at/help/de-de/14-bedienung/66-krediteinkaufe-mit-kundennummer) |
| **Customer payments (AKONTO)** | Customer number → Akonto → amount → Cash → ESC | [AKONTO](https://support.postronik.at/help/de-de/14-bedienung/67-kundeneinzahlungen-mit-akonto) |
| **Customer account + / –** | Change balance without cash (e.g. bank transfer): Customer number → Account – → amount → Account – | [Customer account +/–](https://support.postronik.at/help/de-de/14-bedienung/68-kundenkonto-kundenkonto) |
